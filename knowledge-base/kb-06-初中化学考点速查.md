# 初中化学考点速查

> 按人教版12单元编排，覆盖初中化学全部核心考点，含化学方程式、采分点和易错提醒。

## 第一单元：走进化学世界

### 物理变化与化学变化
- **物理变化**：没有新物质生成（如：水的三态变化、玻璃破碎、蔗糖溶解）
- **化学变化**：有新物质生成（如：燃烧、铁生锈、食物腐败）
- **判断依据**：是否有新物质生成（不是看是否有发光、发热、颜色变化）
- **本质区别**：分子本身是否改变——物理变化分子不变，化学变化分子改变

### 常见仪器与用途
| 仪器 | 用途 | 注意事项 |
|------|------|---------|
| 试管 | 少量试剂反应容器 | 加热时管口不对人，液体不超1/3 |
| 烧杯 | 配制溶液、大量试剂反应 | 加热需垫石棉网 |
| 量筒 | 量取液体体积 | 不能加热、不能配溶液、读数平视凹液面最低处 |
| 酒精灯 | 加热 | 禁止对火、禁止吹灭、用外焰加热 |
| 胶头滴管 | 滴加少量液体 | 竖直悬空、不伸入试管内、用后清洗 |
| 托盘天平 | 称量固体质量 | 左物右码、称量纸、易潮解固体用玻璃器皿 |
| 集气瓶 | 收集气体 | 磨砂面朝下盖玻璃片 |
| 铁架台 | 固定装置 | 铁夹夹在试管中上部 |

### 基本实验操作
- **药品取用**：固体粉末用药匙/纸槽（一斜二送三直立）、块状用镊子（一横二放三慢竖）、液体倾倒（标签向手心、瓶口紧挨试管口）
- **加热**：试管夹从下往上套，夹在试管中上部（距管口约1/3处）
- **气密性检查**：手捂法或热毛巾法——导管口有气泡冒出且松开后水柱回升
- **玻璃仪器洗涤标准**：内壁附着的水既不聚成水滴也不成股流下

## 第二单元：我们周围的空气

### 空气成分（体积分数）
**N₂ ≈ 78%、O₂ ≈ 21%、稀有气体 ≈ 0.94%、CO₂ ≈ 0.03%、其他 ≈ 0.03%**

### 空气中氧气含量测定（红磷燃烧法）
- **原理**：红磷燃烧消耗密闭容器中的O₂，瓶内气压减小，水进入补充
- **方程式**：4P + 5O₂ →点燃→ 2P₂O₅
- **结论**：氧气约占空气体积的1/5
- **关键操作**：红磷过量、装置气密性良好、冷却至室温后打开止水夹
- **不能代替红磷的物质**：木炭（生成CO₂气体）、硫（生成SO₂气体）——因为生成物为气体，瓶内气压不减小

### 氧气的性质
- **物理性质**：无色无味、密度略大于空气、不易溶于水
- **化学性质**：助燃性（不支持燃烧但能助燃）、氧化性
  - 与木炭：C + O₂ →点燃→ CO₂（发出白光）
  - 与硫：S + O₂ →点燃→ SO₂（蓝紫色火焰）
  - 与铁：3Fe + 2O₂ →点燃→ Fe₃O₄（火星四射，瓶底放水或细沙防炸裂）
- **化合反应**：多变一（A + B → AB）

### 氧气的实验室制法
- **高锰酸钾加热法**：2KMnO₄ →△→ K₂MnO₄ + MnO₂ + O₂↑
- **氯酸钾加热法**：2KClO₃ →MnO₂、△→ 2KCl + 3O₂↑（MnO₂作催化剂，需持续加热）
- **过氧化氢分解法**：2H₂O₂ →MnO₂→ 2H₂O + O₂↑
- **催化剂**：一变两不变（改变速率，自身质量和化学性质不变）

### 📌 核心方程式（化合/分解反应）
| 类型 | 方程式 | 反应条件/现象 |
|------|--------|--------------|
| 化合 | 4P + 5O₂ →点燃→ 2P₂O₅ | 大量白烟，放热 |
| 化合 | C + O₂ →点燃→ CO₂ | 发出白光 |
| 化合 | S + O₂ →点燃→ SO₂ | 蓝紫色火焰，刺激气味 |
| 化合 | 3Fe + 2O₂ →点燃→ Fe₃O₄ | 火星四射，黑色固体 |
| 分解 | 2KMnO₄ →△→ K₂MnO₄ + MnO₂ + O₂↑ | 紫黑色→黑色，试管口塞棉花 |
| 分解 | 2KClO₃ →MnO₂、△→ 2KCl + 3O₂↑ | 持续加热，催化剂加速 |
| 分解 | 2H₂O₂ →MnO₂→ 2H₂O + O₂↑ | 常温分解，带火星木条复燃 |

### 气体收集方法选择
| 方法 | 适用条件 | 示例 |
|------|---------|------|
| 排水法 | 不溶于水且不与水反应 | O₂、H₂、CO（不适用CO₂） |
| 向上排空气法 | 密度>空气且不与空气反应 | O₂、CO₂ |
| 向下排空气法 | 密度<空气且不与空气反应 | H₂ |

## 第三单元：物质构成的奥秘

### 分子与原子
- **分子**：保持物质化学性质的最小粒子（同种分子化学性质相同）
- **原子**：化学变化中的最小粒子
- **分子性质**：很小、不断运动、分子间有间隔
- **化学变化本质**：分子分裂为原子，原子重新组合成新分子

### 原子结构
- **原子构成**：原子核（质子+中子）+ 核外电子
- **核电荷数** = 质子数 = 核外电子数 = 原子序数
- **相对原子质量** ≈ 质子数 + 中子数

### 离子
- **阳离子**：原子失去电子（质子数 > 电子数）→ 带正电荷
- **阴离子**：原子得到电子（质子数 < 电子数）→ 带负电荷

### 元素与化学式
- **元素**：质子数（核电荷数）相同的一类原子的总称
- **化合价口诀**：一价钾钠氯氢银，二价氧钙钡镁锌，三铝四硅五价磷
- **化学式书写**：正价在前负价在后，交叉化简

## 第四单元：自然界的水

### 电解水实验
- **方程式**：2H₂O →通电→ 2H₂↑ + O₂↑
- **现象**：两个电极都有气泡，负极气体体积是正极的2倍
- **口诀**：正氧负氢，氢二氧一
- **加入稀硫酸目的**：增强水的导电性

### 水的净化
净化程度由低到高：**沉淀 → 过滤 → 吸附 → 蒸馏**
- **过滤**：一贴二低三靠，玻璃棒引流
- **硬水软化**：煮沸（生活）、蒸馏（实验室）
- **鉴别硬水软水**：加肥皂水——泡沫多为软水，浮渣多为硬水

### 📌 核心方程式（分解/化合反应）
| 类型 | 方程式 | 反应条件/现象 |
|------|--------|--------------|
| 分解 | 2H₂O →通电→ 2H₂↑ + O₂↑ | 正氧负氢，体积比 O₂:H₂ = 1:2 |
| 化合 | 2H₂ + O₂ →点燃→ 2H₂O | 淡蓝色火焰，生成水雾 |

## 第五单元：化学方程式

### 质量守恒定律
参加化学反应的各物质质量总和 = 反应后生成的各物质质量总和

### 微观解释
化学反应前后：原子种类不变、原子数目不变、原子质量不变（三不变）

### 化学方程式书写
1. 写化学式
2. 配平（最小公倍数法/奇偶配平法/观察法）
3. 注明反应条件和生成物状态（↑气体↓沉淀）

### 根据化学方程式的计算
六步走：设→写→标→列→求→答

### 📌 核心方程式（四大基本反应类型示例）
| 类型 | 方程式 | 关联实验 |
|------|--------|---------|
| 化合 | 2H₂ + O₂ →点燃→ 2H₂O | 氢气燃烧 |
| 分解 | 2H₂O →通电→ 2H₂↑ + O₂↑ | 电解水 |
| 置换 | Fe + CuSO₄ → FeSO₄ + Cu | 金属活动性 |
| 复分解 | HCl + NaOH → NaCl + H₂O | 中和反应 |

## 第六单元：碳和碳的氧化物

### 碳的单质
| 单质 | 结构 | 性质 | 用途 |
|------|------|------|------|
| 金刚石 | 正四面体网状 | 最硬 | 切割玻璃、钻头 |
| 石墨 | 层状结构 | 导电、润滑 | 电极、润滑剂、铅笔芯 |
| C₆₀ | 足球状分子 | 特殊 | 材料科学 |

### 碳的化学性质
- **常温稳定性**：古代字画千年不褪色
- **可燃性**：C + O₂ →点燃→ CO₂（充分燃烧）/ 2C + O₂ →点燃→ 2CO（不充分燃烧）
- **还原性**：C + 2CuO →高温→ 2Cu + CO₂↑（黑色固体变红，澄清石灰水变浑浊）

### 碳的氧化物——CO₂
- **实验室制法**：CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑
- **验满**：燃着木条放瓶口熄灭
- **检验**：通入澄清石灰水变浑浊 Ca(OH)₂ + CO₂ → CaCO₃↓ + H₂O
- **与水反应**：CO₂ + H₂O → H₂CO₃（碳酸使石蕊变红）
- **不支持燃烧也不可燃烧**（一般情况）
- **温室效应**：CO₂过多排放导致全球变暖

### 碳的氧化物——CO
- **物理性质**：无色无味、密度略小于空气、难溶于水
- **可燃性**：2CO + O₂ →点燃→ 2CO₂（蓝色火焰）
- **还原性**：CO + CuO →△→ Cu + CO₂
- **毒性**：CO与血红蛋白结合力是O₂的200-300倍，造成缺氧中毒

### 📌 核心方程式（碳及其氧化物）
| 类型 | 方程式 | 反应条件/现象 |
|------|--------|--------------|
| 化合（可燃） | C + O₂ →点燃→ CO₂ | 充分燃烧，发白光 |
| 化合（可燃） | 2C + O₂ →点燃→ 2CO | 不充分燃烧 |
| 化合（可燃） | 2CO + O₂ →点燃→ 2CO₂ | 蓝色火焰 |
| 化合（与水） | CO₂ + H₂O → H₂CO₃ | 石蕊变红 |
| 复分解 | CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑ | 实验室制 CO₂ |
| 复分解 | Ca(OH)₂ + CO₂ → CaCO₃↓ + H₂O | 澄清石灰水变浑浊 |
| 置换（还原） | C + 2CuO →高温→ 2Cu + CO₂↑ | 黑色变红色 |
| 置换（还原） | CO + CuO →△→ Cu + CO₂ | 黑色变红色 |
| 分解（工业） | CaCO₃ →高温→ CaO + CO₂↑ | 工业制生石灰 |
| 分解（不稳定酸） | H₂CO₃ → H₂O + CO₂↑ | 碳酸不稳定 |

## 第七单元：燃料及其利用

### 燃烧三条件（缺一不可）
1. 可燃物
2. 氧气（或空气）
3. 温度达到着火点

### 灭火原理（破坏三条件之一）
1. 清除可燃物（如：关闭燃气阀门）
2. 隔绝氧气（如：CO₂灭火器、锅盖盖灭油锅火）
3. 降温至着火点以下（如：水灭火）

### 化石燃料
煤、石油、天然气——混合物、不可再生能源

### 清洁能源
- **CH₄**（天然气主要成分）：CH₄ + 2O₂ →点燃→ CO₂ + 2H₂O
- **C₂H₅OH**（乙醇）：C₂H₅OH + 3O₂ →点燃→ 2CO₂ + 3H₂O（可再生能源）
- **H₂**（最清洁燃料）：2H₂ + O₂ →点燃→ 2H₂O（热值高、无污染、原料丰富）

### 📌 核心方程式（燃烧反应）
| 类型 | 方程式 | 反应条件/现象 |
|------|--------|--------------|
| 燃烧 | CH₄ + 2O₂ →点燃→ CO₂ + 2H₂O | 天然气燃烧，蓝色火焰 |
| 燃烧 | C₂H₅OH + 3O₂ →点燃→ 2CO₂ + 3H₂O | 乙醇燃烧，淡蓝色火焰 |
| 燃烧 | 2H₂ + O₂ →点燃→ 2H₂O | 氢气燃烧，淡蓝色火焰 |
| 燃烧 | 2CO + O₂ →点燃→ 2CO₂ | 一氧化碳燃烧，蓝色火焰 |

## 第八单元：金属和金属材料

### 金属共性
金属光泽、导电性、导热性、延展性

### 合金
- **概念**：金属+金属/非金属熔合而成的具有金属特性的混合物
- **特性**：合金熔点一般低于成分金属，硬度一般更大
- **生铁与钢**：含碳量不同（生铁2%-4.3%、钢0.03%-2%）

### 金属活动性顺序
**K > Ca > Na > Mg > Al > Zn > Fe > Sn > Pb > (H) > Cu > Hg > Ag > Pt > Au**

应用规则：
1. 前置后（排在前面的金属能把后面的金属从其盐溶液中置换出来）
2. K/Ca/Na除外（太活泼，先与水反应）
3. 盐溶液必须可溶

### 铁的锈蚀与防护
- **条件**：铁 + O₂ + H₂O → 铁锈（Fe₂O₃·xH₂O）
- **特点**：铁锈疏松多孔，不能保护内部铁继续生锈
- **防护**：干燥、涂油、刷漆、镀金属（镀锌/镀铬）、制成合金（不锈钢）

### 📌 核心方程式（置换反应为主）
| 类型 | 方程式 | 反应条件/现象 |
|------|--------|--------------|
| 金属+酸 | Mg + 2HCl → MgCl₂ + H₂↑ | 剧烈产生气泡 |
| 金属+酸 | 2Al + 6HCl → 2AlCl₃ + 3H₂↑ | 剧烈产生气泡 |
| 金属+酸 | Zn + H₂SO₄ → ZnSO₄ + H₂↑ | 实验室制氢气首选 |
| 金属+酸 | Fe + 2HCl → FeCl₂ + H₂↑ | 缓慢产生气泡，溶液变浅绿色 |
| 金属+酸 | Fe + H₂SO₄ → FeSO₄ + H₂↑ | 缓慢产生气泡 |
| 金属+盐 | Fe + CuSO₄ → FeSO₄ + Cu | 铁表面附红色固体，溶液蓝→浅绿 |
| 金属+盐 | Cu + 2AgNO₃ → Cu(NO₃)₂ + 2Ag | 铜表面附银白色固体，溶液变蓝 |
| 金属+盐 | 2Al + 3CuSO₄ → Al₂(SO₄)₃ + 3Cu | 铝表面附红色固体 |
| 还原（炼铁） | 3CO + Fe₂O₃ →高温→ 2Fe + 3CO₂ | 高炉炼铁原理，红色固体变黑 |
| 金属氧化物+酸 | Fe₂O₃ + 3H₂SO₄ → Fe₂(SO₄)₃ + 3H₂O | 除铁锈，溶液变黄色 |

### ❌ 易错提醒
- 铁与酸/盐反应生成 **+2 价亚铁**（FeCl₂/FeSO₄），溶液为浅绿色，不是 +3 价
- K/Ca/Na 太活泼，不能用于置换盐溶液中的金属（先与水反应）
- 铜不能与稀盐酸/稀硫酸反应（位于氢后）

## 第九单元：溶液

### 溶液
- **定义**：一种或几种物质分散到另一种物质中形成的均一、稳定的混合物
- **特征**：均一性、稳定性、混合物
- **溶解放热**：NaOH固体溶于水放热
- **溶解吸热**：NH₄NO₃固体溶于水吸热
- **温度基本不变**：NaCl固体溶于水温度基本不变

### 溶解度
- **定义**：在一定温度下，某固态物质在100g溶剂中达到饱和状态时所溶解的质量（单位：g）
- **四要素**：一定温度、100g溶剂、饱和状态、质量/g
- **气体溶解度**：温度越高溶解度越小、压强越大溶解度越大

### 溶质质量分数
- **公式**：ω = (m溶质 / m溶液) × 100%
- **稀释公式**：m浓 × ω浓 = m稀 × ω稀（稀释前后溶质质量不变）

### 📌 核心方程式（溶液中反应/结晶）
| 类型 | 方程式 | 反应条件/现象 |
|------|--------|--------------|
| 结晶水合物受热 | CuSO₄·5H₂O →△→ CuSO₄ + 5H₂O | 蓝色晶体变白色粉末 |
| 水合反应 | CuSO₄ + 5H₂O → CuSO₄·5H₂O | 白色粉末变蓝色，放热 |
| 复分解（溶液） | Ca(OH)₂ + Na₂CO₃ → CaCO₃↓ + 2NaOH | 工业制烧碱（苛化法） |
| 复分解（溶液） | CaCl₂ + Na₂CO₃ → CaCO₃↓ + 2NaCl | 溶液中产生白色沉淀 |

### 溶解度曲线（中考重点）

#### 曲线定义
在直角坐标系中描述**某物质溶解度随温度变化**的关系曲线。横坐标为温度（℃），纵坐标为溶解度（g/100g 水）。

#### 五大读取要点
1. **某温度下溶解度**：过该温度作垂线，交点的纵坐标即为该温度的溶解度
2. **同温度下比较大小**：上方的曲线代表溶解度更大
3. **变化趋势判断**：曲线**陡**（斜率大）→ 溶解度受温度影响大；曲线**平缓**（斜率小）→ 受温度影响小
4. **曲线交点**：该温度下两物质溶解度**相等**
5. **曲线上方/下方点**：上方点为饱和溶液（有过剩固体），下方点为不饱和溶液

#### 三种典型曲线类型
| 类型 | 代表物质 | 曲线特征 | 结晶方法 |
|------|----------|----------|----------|
| **陡升型** | KNO₃、NaNO₃ | 随温度升高溶解度急剧增大 | **冷却结晶**（降温） |
| **缓升型** | NaCl | 随温度升高溶解度变化很小 | **蒸发结晶** |
| **下降型** | Ca(OH)₂、气体 | 随温度升高溶解度反而减小 | **升温结晶**（特殊）|

### 结晶方法判断

#### 核心口诀
> **"陡用降温，缓用蒸发"**
> 曲线**陡**（受温度影响大）→ 冷却热饱和溶液（降温结晶）
> 曲线**平缓**（受温度影响小）→ 蒸发结晶

#### 判断流程
1. 查物质溶解度曲线类型（陡升/缓升/下降）
2. 选择对应结晶方法
3. 应用实例：
   - **NaCl**（缓升）→ 海水晒盐用蒸发结晶
   - **KNO₃**（陡升）→ 分离 KNO₃ 和 NaCl 混合物：冷却结晶先析出 KNO₃
   - **Na₂CO₃·10H₂O**（陡升）→ "冬天捞碱"用冷却结晶

### 📌 中考高频考点
1. **从混合物中分离**：KNO₃（含少量 NaCl）→ 冷却结晶；NaCl（含少量 KNO₃）→ 蒸发结晶
2. **等质量溶剂中析出量比较**：降温相同温度，曲线陡的物质析出更多
3. **交点温度判断**：两曲线交点表示溶解度相等的温度
4. **不饱和→饱和转化**：陡升型加溶质/蒸发溶剂/降温；缓升型（NaCl）只能加溶质/蒸发

### ❌ 易错提醒
- **NaCl 不能用冷却结晶**：溶解度随温度变化小，降温析出量极少
- **Ca(OH)₂ 是反常曲线**：温度升高溶解度反而减小（与大多数固体相反）
- **气体溶解度也随温度升高而减小**：加热汽水冒泡就是这个原理
- **"冬天捞碱"指 Na₂CO₃·10H₂O**（结晶水合物），不是 NaOH

## 第十单元：酸和碱

### 酸碱指示剂
| 指示剂 | 酸性 | 中性 | 碱性 |
|--------|------|------|------|
| 石蕊 | 红色 | 紫色 | 蓝色 |
| 酚酞 | 无色 | 无色 | 红色 |

### 常见酸
**盐酸（HCl）**、**硫酸（H₂SO₄）**、**硝酸（HNO₃）**

### 浓硫酸特性
1. **吸水性**：用作干燥剂（干燥O₂、CO₂等，不能干燥NH₃）
2. **脱水性**：使有机物碳化（蔗糖变黑实验）
3. **溶于水放出大量热**：稀释时酸入水，不断搅拌

### 酸的化学通性
1. 与指示剂反应（石蕊变红、酚酞不变色）
2. 与活泼金属反应生成盐+H₂（位于H前的金属）
3. 与金属氧化物反应生成盐+水
4. 与碱反应（中和反应）生成盐+水
5. 与某些盐反应生成新酸+新盐

### 碱的化学通性
1. 与指示剂反应（石蕊变蓝、酚酞变红）
2. 与非金属氧化物反应生成盐+水
3. 与酸反应（中和反应）
4. 与某些盐反应生成新碱+新盐

### 中和反应
- **定义**：酸+碱→盐+水
- **实质**：H⁺ + OH⁻ → H₂O
- **应用**：改良酸性土壤（熟石灰）、处理酸性废水、胃药（Al(OH)₃）

### pH
- pH < 7 酸性，pH = 7 中性，pH > 7 碱性
- pH越小酸性越强，pH越大碱性越强

### 📌 核心方程式——酸的通性（5 类）
| 通性 | 方程式 | 现象/用途 |
|------|--------|----------|
| ①酸+指示剂 | 紫色石蕊遇酸变红、酚酞遇酸不变色 | 酸碱性强弱判断 |
| ②酸+活泼金属 | Zn + H₂SO₄ → ZnSO₄ + H₂↑ | 实验室制氢气 |
| ②酸+活泼金属 | Fe + 2HCl → FeCl₂ + H₂↑ | 溶液变浅绿色 |
| ③酸+金属氧化物 | Fe₂O₃ + 6HCl → 2FeCl₃ + 3H₂O | 除铁锈，溶液变黄色 |
| ③酸+金属氧化物 | CuO + H₂SO₄ → CuSO₄ + H₂O | 黑色溶解，溶液变蓝 |
| ④酸+碱（中和） | HCl + NaOH → NaCl + H₂O | 无明显现象，放热 |
| ④酸+碱（中和） | H₂SO₄ + 2NaOH → Na₂SO₄ + 2H₂O | 中和反应 |
| ④酸+碱（中和） | HCl + Ca(OH)₂ → CaCl₂ + 2H₂O | 中和反应 |
| ⑤酸+盐 | HCl + AgNO₃ → AgCl↓ + HNO₃ | 白色沉淀，检验 Cl⁻ |
| ⑤酸+盐 | H₂SO₄ + BaCl₂ → BaSO₄↓ + 2HCl | 白色沉淀，检验 SO₄²⁻ |
| ⑤酸+盐 | CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑ | 实验室制 CO₂ |

### 📌 核心方程式——碱的通性（4 类）
| 通性 | 方程式 | 现象/用途 |
|------|--------|----------|
| ①碱+指示剂 | 紫色石蕊遇碱变蓝、酚酞遇碱变红 | 碱性强弱判断 |
| ②碱+非金属氧化物 | 2NaOH + CO₂ → Na₂CO₃ + H₂O | 吸收 CO₂（防变质） |
| ②碱+非金属氧化物 | 2NaOH + SO₂ → Na₂SO₃ + H₂O | 吸收 SO₂（处理废气） |
| ②碱+非金属氧化物 | Ca(OH)₂ + CO₂ → CaCO₃↓ + H₂O | 澄清石灰水变浑浊，检验 CO₂ |
| ③碱+酸（中和） | NaOH + HCl → NaCl + H₂O | 同酸通性④ |
| ④碱+盐 | CuSO₄ + 2NaOH → Cu(OH)₂↓ + Na₂SO₄ | 蓝色沉淀 |
| ④碱+盐 | FeCl₃ + 3NaOH → Fe(OH)₃↓ + 3NaCl | 红褐色沉淀 |
| ④碱+盐 | Na₂CO₃ + Ca(OH)₂ → CaCO₃↓ + 2NaOH | 工业制烧碱 |

### ❌ 易错提醒
- 铁与酸/盐反应生成 **+2 价亚铁**（浅绿色溶液），不是 +3 价（黄色）
- 氢氧化钠必须密封保存：易吸水潮解 + 易吸收 CO₂ 变质（Na₂CO₃）
- 氢氧化钙需密封保存：吸收 CO₂ 变质（CaCO₃）
- 中和反应不一定有明显现象，常借助指示剂判断
- 稀释浓硫酸：**酸入水**，沿器壁慢慢倒入，不断搅拌

## 第十一单元：盐 化肥

### 常见盐
| 化学式 | 名称 | 俗称 | 用途 |
|--------|------|------|------|
| NaCl | 氯化钠 | 食盐 | 调味、防腐、制生理盐水 |
| Na₂CO₃ | 碳酸钠 | 纯碱/苏打 | 制玻璃、造纸、洗涤 |
| NaHCO₃ | 碳酸氢钠 | 小苏打 | 发酵粉、治疗胃酸过多 |
| CaCO₃ | 碳酸钙 | 石灰石/大理石 | 建筑材料、制CO₂ |

### 复分解反应
- **定义**：AB + CD → AD + CB
- **发生条件**：生成物中有沉淀/气体/水之一即可
- **口诀**：酸碱中和一定行，盐盐反应皆可溶，酸盐反应盐可溶且生成物有沉淀/气体

### 化学肥料
- **氮肥（N）**：促生长、叶色浓绿
- **磷肥（P）**：促根系发育、增强抗寒抗旱
- **钾肥（K）**：促茎秆粗壮、增强抗倒伏抗病虫害
- **铵态氮肥检验**：加碱加热，产生使湿润红色石蕊试纸变蓝的气体（NH₃）

### 常见化肥物质清单（中考必记）

#### 氮肥（促茎叶生长，合成蛋白质）
| 名称 | 化学式 | 相对分子质量 | 含氮量 | 备注 |
|------|--------|:-----------:|:------:|------|
| **尿素** | CO(NH₂)₂ | 60 | **46.7%** | **含氮量最高的固体氮肥**，中性，适用各类土壤 |
| 硝酸铵 | NH₄NO₃ | 80 | 35% | 铵态+硝态氮，受热/撞击易爆炸 |
| 硫酸铵 | (NH₄)₂SO₄ | 132 | 21.2% | 铵态氮肥，长期使用使土壤酸化板结 |
| 碳酸氢铵 | NH₄HCO₃ | 79 | 17.7% | 铵态氮肥，易分解（NH₃↑+CO₂↑+H₂O），俗称气肥 |
| 氯化铵 | NH₄Cl | 53.5 | 26.2% | 铵态氮肥，忌氯作物慎用 |
| 硝酸钠 | NaNO₃ | 85 | 16.5% | 硝态氮肥 |

#### 磷肥（促根系发育、开花结果、抗寒抗旱）
| 名称 | 化学式 | 俗称 | 备注 |
|------|--------|------|------|
| **过磷酸钙（普钙）** | **Ca(H₂PO₄)₂·CaSO₄** | 普钙 | **混合物**，含磷酸二氢钙+硫酸钙 |
| 重过磷酸钙（重钙） | Ca(H₂PO₄)₂ | 重钙 | 有效磷含量比普钙高 |
| 钙镁磷肥 | Ca₃(PO₄)₂ | — | 难溶性磷肥 |
| 磷矿粉 | Ca₃(PO₄)₂ | — | 不溶于水，迟效磷肥 |

#### 钾肥（促茎秆粗壮、抗倒伏、抗病虫害）
| 名称 | 化学式 | 备注 |
|------|--------|------|
| **硫酸钾** | K₂SO₄ | 适用忌氯作物（烟草、薯类）|
| **氯化钾** | KCl | 常用钾肥，含钾量高，忌氯作物慎用 |
| 硝酸钾 | KNO₃ | **复合肥**（含 K+N）|
| 碳酸钾 | K₂CO₃ | 草木灰主成分，碱性 |

#### 复合肥（含 N、P、K 中两种或以上）
| 名称 | 化学式 | 含量 |
|------|--------|------|
| 磷酸二氢铵 | NH₄H₂PO₄ | N + P |
| 磷酸氢二铵 | (NH₄)₂HPO₄ | N + P |
| 硝酸钾 | KNO₃ | N + K |
| 磷酸二氢钾 | KH₂PO₄ | P + K |

### 含氮量计算示例（必考）
- **尿素 CO(NH₂)₂**：Mr = 60，N% = (2×14)/60 ×100% ≈ **46.7%**
- **硝酸铵 NH₄NO₃**：Mr = 80，N% = (2×14)/80 ×100% = **35%**
- **结论**：尿素含氮量最高，等质量下提供的氮元素最多

### 铵态氮肥使用禁忌
- ❌ **不能与碱性物质混合施用**（如草木灰、熟石灰）：发生反应放出 NH₃，降低肥效
  - (NH₄)₂SO₄ + Ca(OH)₂ → CaSO₄ + 2NH₃↑ + 2H₂O
- ❌ **不能在烈日下施用**：NH₃ 易挥发（尤其 NH₄HCO₃ 俗称"气肥"）
- ✅ **深施覆土**：减少 NH₃ 挥发

### 化肥鉴别方法
| 步骤 | 操作 | 现象 |
|------|------|------|
| 1 | 观外观 | 白色晶体（氮肥/钾肥）/ 灰褐色粉末（磷肥）|
| 2 | 闻气味 | NH₄HCO₃ 有氨味 |
| 3 | 加熟石灰研磨 | 铵态氮肥有刺激性氨味（NH₃）|
| 4 | 加水溶解 | 氮肥/钾肥可溶；磷肥大多难溶 |

### 📌 核心方程式（复分解反应为主）

### 📌 核心方程式（复分解反应为主）
| 类型 | 方程式 | 现象/用途 |
|------|--------|----------|
| 酸+盐（Cl⁻检验） | HCl + AgNO₃ → AgCl↓ + HNO₃ | 白色沉淀，不溶于稀硝酸 |
| 盐+盐（Cl⁻检验） | NaCl + AgNO₃ → AgCl↓ + NaNO₃ | 白色沉淀，不溶于稀硝酸 |
| 酸+盐（SO₄²⁻检验） | H₂SO₄ + BaCl₂ → BaSO₄↓ + 2HCl | 白色沉淀，不溶于稀硝酸 |
| 盐+盐（SO₄²⁻检验） | Na₂SO₄ + Ba(NO₃)₂ → BaSO₄↓ + 2NaNO₃ | 白色沉淀，不溶于稀硝酸 |
| 酸+碳酸盐 | Na₂CO₃ + 2HCl → 2NaCl + H₂O + CO₂↑ | 产生气泡，检验 CO₃²⁻ |
| 酸+碳酸盐 | K₂CO₃ + 2HCl → 2KCl + H₂O + CO₂↑ | 产生气泡 |
| 碱+盐（工业制碱） | Na₂CO₃ + Ca(OH)₂ → CaCO₃↓ + 2NaOH | 产生白色沉淀 |
| 铵盐检验 | NH₄Cl + NaOH →△→ NaCl + H₂O + NH₃↑ | 产生刺激性气味气体 |
| 高温分解（工业） | CaCO₃ →高温→ CaO + CO₂↑ | 工业制生石灰（CaO） |

### ❌ 易错提醒
- 检验 Cl⁻ 用 AgNO₃ + 稀硝酸；检验 SO₄²⁻ 用 BaCl₂/Ba(NO₃)₂ + 稀硝酸
- AgCl 和 BaSO₄ 是既不溶于水也不溶于酸的沉淀（其他白色沉淀溶于酸）
- 碳酸盐（CO₃²⁻）检验：加稀盐酸，产生气体通入澄清石灰水变浑浊
- 铵态氮肥不能与碱性物质混合施用（产生 NH₃ 逸出，降低肥效）

## 第十二单元：化学与生活

### 六大营养素
蛋白质、糖类、油脂、维生素、无机盐、水

### 有机物与无机物
- **有机物**：含碳元素的化合物（CO、CO₂、碳酸盐等少数除外）
- **有机高分子材料**：天然（棉花/羊毛/蚕丝/天然橡胶）vs 合成（塑料/合成纤维/合成橡胶）
- **白色污染**：废弃塑料造成的环境污染

### 人体元素
- **常量元素**：C、H、O、N、Ca、P、K、S、Na、Cl、Mg（含量>0.01%）
- **微量元素**：Fe（贫血）、Zn（发育迟缓）、Se（抗氧化）、I（甲状腺肿大）、F（龋齿）

### 环境污染
- **酸雨**：SO₂、NO₂排放 → pH<5.6
- **温室效应**：CO₂过量排放
- **臭氧层破坏**：氟利昂（CFCs）

### 📌 核心方程式（有机/生物反应）

|------|--------|----------|

| 燃烧（乙醇） | C₂H₅OH + 3O₂ →点燃→ 2CO₂ + 3H₂O | 生物质能源 |

## 附录 B：化学之三（初中必背 42 条）

### B.1 三种基本化学反应类型（不含复分解）
- **化合反应**：A + B → AB（多变一）
- **分解反应**：AB → A + B（一变多）
- **置换反应**：A + BC → AC + B（单换单）
> 复分解反应为第四种基本类型：AB + CD → AD + CB（双交换）

### B.2 三种常见还原剂
- **C（碳）**、**CO（一氧化碳）**、**H₂（氢气）**
- 共性：高温下能夺取金属氧化物中的氧

### B.3 三种常见氧化剂
- **O₂（氧气）**、**CO₂（二氧化碳，部分反应）**、**H₂O₂（过氧化氢，实验室制氧）**

### B.4 构成物质的三种微粒
- **分子**（如 H₂O、O₂）、**原子**（如 Fe、Cu）、**离子**（如 Na⁺、Cl⁻）

### B.5 三种可燃气体（H₂ + CO + CH₄）
- **H₂**：淡蓝色火焰，产物 H₂O
- **CO**：蓝色火焰，产物 CO₂
- **CH₄**：蓝色火焰，产物 CO₂ + H₂O
> 鉴别：燃烧后用干冷烧杯罩住——内壁有无水雾区分是否含 H；通入澄清石灰水变浑浊区分是否含 C。

### B.6 三大化石燃料
- **煤**（工业的粮食）、**石油**（工业的血液）、**天然气**（主要成分 CH₄）
> 均为**混合物** + **不可再生能源**

### B.7 三种常见的金属矿物
- **赤铁矿**（Fe₂O₃）、**磁铁矿**（Fe₃O₄）、**菱铁矿**（FeCO₃）

### B.8 炼铁高炉三原料
- **铁矿石** + **焦炭** + **石灰石**

### B.9 三大合成材料
- **塑料**（聚乙烯）、**合成纤维**（涤纶/锦纶）、**合成橡胶**
> 棉花、羊毛、天然橡胶为天然材料

### B.10 三大酸性氧化物
- **CO₂**、**SO₂**、**SO₃**（溶于水成对应含氧酸）

### B.11 三大可溶性碳酸盐
- **Na₂CO₃**、**K₂CO₃**、**(NH₄)₂CO₃**

### B.12 铁的三种氧化物
- **FeO**（黑色）、**Fe₂O₃**（红棕色，赤铁矿主成分）、**Fe₃O₄**（黑色，磁铁矿主成分）

### B.13 碳的三种同素异形体
- **金刚石**（天然最硬）、**石墨**（导电润滑）、**C₆₀**（足球烯，纳米材料）

### B.14 三种不溶性碱（颜色记忆）
- **Cu(OH)₂**（蓝色）、**Fe(OH)₃**（红褐色）、**Mg(OH)₂**（白色）

### B.15 三种不溶于稀硝酸的白色沉淀
- **BaSO₄**（硫酸钡）、**AgCl**（氯化银）、**AgBr / AgI**（淡黄/黄色）

### B.16 三种溶质质量分数计算公式
- **定义式**：ω = m(质) / m(液) × 100%
- **稀释式**：m(浓) × ω(浓) = m(稀) × ω(稀)
- **稀释加水量**：m(水) = m(稀) − m(浓)

### B.17 三种常见的氮肥
- **NH₄HCO₃**（碳酸氢铵/气肥）、**(NH₄)₂SO₄**（硫酸铵）、**CO(NH₂)₂**（尿素，含氮最高 46.7%）

### B.18 三种常见的复合肥
- **KNO₃**（硝酸钾）、**NH₄H₂PO₄**（磷酸二氢铵）、**(NH₄)₂HPO₄**（磷酸氢二铵）

### B.19 铁的三种化学性质（与三类物质反应）
- 与**氧气**反应：3Fe + 2O₂ →点燃→ Fe₃O₄（火星四射，黑色固体）
- 与**酸**反应：Fe + 2HCl → FeCl₂ + H₂↑（生成浅绿色 Fe²⁺ 溶液）
- 与**盐溶液**反应：Fe + CuSO₄ → FeSO₄ + Cu（湿法炼铜）

### B.20 三大环境问题
- **酸雨**（SO₂/NO₂ → pH<5.6）
- **温室效应**（CO₂ 增多）
- **白色污染**（废弃塑料制品）

### B.21 实验室制取气体三步骤
- **装置气密性检查** → **装药品固定装置** → **收集气体验满**

### B.22 实验室制取 CO₂ 三不用
- **不用稀硫酸**（生成微溶 CaSO₄ 阻止反应）
- **不用浓盐酸**（HCl 易挥发使 CO₂ 不纯）
- **不用碳酸钠粉末**（反应过快，不便控制）

### B.23 三种常见试纸
- **pH 试纸**（测酸碱度）、**红色石蕊试纸**（遇碱变蓝）、**蓝色石蕊试纸**（遇酸变红）

### B.24 三种常见催化剂（初中范围）
- **MnO₂**（催化 KClO₃ 或 H₂O₂ 分解）、**V₂O₅**（接触法制硫酸）、**Fe**（催化氨合成）

> 📌 「化学之三」是中考选择/填空的高频考点，建议每条都能完整背诵 + 举例。

## 附录 E：常见物质除杂方法（16 条，中考必考）

> **除杂原则**：① 不增（不引入新杂质）② 不减（不消耗主体物质）③ 易分（生成物与主体易分离）④ 简便（操作简单可行）

### E.1 气体除杂

| # | 主体气体 | 杂质 | 除杂试剂 / 方法 | 化学原理 |
|---|---------|------|----------------|---------|
| 1 | **CO₂** | HCl（氯化氢）| 饱和 NaHCO₃ 溶液洗气 | NaHCO₃ + HCl → NaCl + H₂O + CO₂↑（不消耗 CO₂） |
| 2 | **CO₂** | H₂O（水蒸气）| 浓硫酸洗气 | 浓硫酸吸水性 |
| 3 | **CO** | CO₂ | 通过 NaOH 溶液洗气 | 2NaOH + CO₂ → Na₂CO₃ + H₂O |
| 4 | **H₂** | HCl | 通过 NaOH 溶液洗气 | HCl + NaOH → NaCl + H₂O |
| 5 | **O₂** | 水蒸气 | 通过干燥管（无水 CaCl₂ 或浓硫酸）| 吸水性 |
| 6 | **CO₂** | SO₂ | 饱和 NaHCO₃ 溶液 + 品红溶液检验 | 与 HCl 除杂同理 |

### E.2 固体/溶液除杂

| # | 主体物质 | 杂质 | 除杂试剂 / 方法 | 化学原理 |
|---|---------|------|----------------|---------|
| 7 | **CuSO₄ 溶液** | H₂SO₄（少量）| 加入过量 CuO 粉末，过滤 | CuO + H₂SO₄ → CuSO₄ + H₂O |
| 8 | **NaCl 溶液** | Na₂CO₃ | 加入适量稀 HCl | Na₂CO₃ + 2HCl → 2NaCl + H₂O + CO₂↑ |
| 9 | **NaOH 溶液** | Na₂CO₃ | 加入适量 Ca(OH)₂ 溶液，过滤 | Na₂CO₃ + Ca(OH)₂ → CaCO₃↓ + 2NaOH |
| 10 | **CaCl₂ 溶液** | HCl | 加入过量 CaCO₃ 粉末，过滤 | CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑ |
| 11 | **FeSO₄ 溶液** | CuSO₄ | 加入过量铁粉，过滤 | Fe + CuSO₄ → FeSO₄ + Cu |
| 12 | **Cu 粉** | Fe 粉 | 加入足量稀 HCl 或稀 H₂SO₄，过滤 | Fe + 2HCl → FeCl₂ + H₂↑（Cu 不反应）|
| 13 | **CuO（固体）** | Cu | 在 O₂ 流中加热 | 2Cu + O₂ →△→ 2CuO |
| 14 | **CaO** | CaCO₃ | 高温煅烧 | CaCO₃ →高温→ CaO + CO₂↑ |
| 15 | **NaCl 固体** | 泥沙 | 溶解 → 过滤 → 蒸发结晶 | NaCl 易溶，泥沙不溶 |
| 16 | **KNO₃ 固体** | NaCl（少量）| 溶解 → 冷却结晶（降温）→ 过滤 | KNO₃ 溶解度随温度变化大，NaCl 变化小 |

> 📌 **技巧**：金属除杂——「**前置后换前**」（金属活动性顺序中前面的金属可以从盐溶液中置换后面的金属）。

## 附录 F：有毒物质一览（实验安全必考）

### F.1 有毒固体

| 物质 | 化学式 | 毒性部位 | 安全措施 |
|------|-------|---------|---------|
| 白磷 | P₄ | 剧毒，误食 0.1 g 致死 | 保存在水中，严禁手触 |
| 红磷 | P | 毒性低，但粉尘吸入有害 | 密封保存 |
| 铅盐 | Pb²⁺（Pb(NO₃)₂ 等）| 神经毒性、骨骼沉积 | 戴手套，废液集中回收 |
| 钡盐（可溶） | BaCl₂、Ba(NO₃)₂ | Ba²⁺ 剧毒，致心律失常 | BaSO₄（不溶）无毒，可作钡餐 |
| 汞 | Hg | 汞蒸气损害神经、肾 | 水银温度计破损立即撒硫粉覆盖（Hg + S → HgS）|
| 亚硝酸钠 | NaNO₂ | 误食致高铁血红蛋白症，0.2 g 致死 | 严禁作食盐替代 |
| 氰化物 | KCN / NaCN | 剧毒，0.05 g 致死 | 严禁进入实验室 |
| 发烟硫酸 | H₂SO₄·SO₃ | 强腐蚀 + 强氧化 | 戴护目镜，严禁身体接触 |

### F.2 有毒液体

| 物质 | 化学式 | 危害 |
|------|-------|------|
| 甲醇 | CH₃OH | 饮用致盲，工业酒精含 4% 甲醇 |
| 甲醛 | HCHO | 致癌，刺激呼吸道（室内装修污染源）|
| 苯 | C₆H₆ | 致癌，损伤造血系统 |
| 浓硫酸 | H₂SO₄ | 强腐蚀 + 强脱水 |
| 浓硝酸 | HNO₃ | 强腐蚀 + 强氧化 |
| 氢氟酸 | HF | 腐蚀玻璃，损伤骨骼 |

### F.3 有毒气体

| 物质 | 化学式 | 毒性 | 实验室防范 |
|------|-------|------|-----------|
| 一氧化碳 | CO | 与血红蛋白结合（结合力是 O₂ 的 250 倍），缺氧致死 | 尾气点燃或收集处理 |
| 二氧化硫 | SO₂ | 刺激呼吸道，形成酸雨 | 尾气通过 NaOH 溶液 |
| 二氧化氮 | NO₂ | 刺激肺，形成光化学烟雾 | 尾气通过 NaOH 溶液 |
| 硫化氢 | H₂S | 剧毒，臭鸡蛋味，损伤神经系统 | 通风橱内操作 |
| 氯气 | Cl₂ | 剧毒，损伤呼吸道 | 通风橱 + 尾气 NaOH 吸收 |
| 氯化氢 | HCl | 刺激呼吸道（浓时腐蚀）| 通风橱操作 |
| 氨气 | NH₃ | 刺激眼、呼吸道 | 通风 + 尾气稀硫酸吸收 |

> 🔴 **安全红线**：① 上述气体实验必须在**通风橱**或**尾气处理装置**中进行；② 严禁徒手接触白磷、汞、钡盐、铅盐；③ 误食重金属盐——立即喝大量牛奶/生鸡蛋清（蛋白质与重金属盐结合减少吸收）。

# 居家安全拓展实验库

> 无腐蚀性、无易燃易爆试剂的生活化探究实验，配套家用替代器材、安全操作规范和居家禁止事项。

## HOME001 · 小苏打与白醋反应制CO₂

### 实验对应
课标实验：实验室制取二氧化碳（石灰石+稀盐酸）

### 家用器材替代
| 实验室器材 | 家用替代 |
|-----------|---------|
| 锥形瓶 | 矿泉水瓶 |
| 石灰石（CaCO₃） | 小苏打（NaHCO₃） |
| 稀盐酸（HCl） | 白醋（CH₃COOH） |

### 化学反应
**NaHCO₃ + CH₃COOH → CH₃COONa + H₂O + CO₂↑**

### 安全操作规范
1. 实验应在通风处（厨房或阳台）进行，家长需在场陪同
2. 使用矿泉水瓶不要拧紧瓶盖（CO₂产生会导致压力增大）
3. 产生的气体不要对着人脸
4. 小苏打和白醋用量不要太多（少量即可观察现象）
5. 可以在瓶口套一个气球观察气体产生

### 🚫 居家禁止事项
- 禁止使用密封容器（有爆裂风险）
- 禁止将气体对着人脸
- 禁止使用浓度过高的醋酸（可能刺激皮肤）

## HOME002 · 紫甘蓝自制酸碱指示剂

### 实验对应
课标实验：酸碱指示剂（石蕊、酚酞）鉴别酸碱性

### 家用器材替代
| 实验室器材 | 家用替代 |
|-----------|---------|
| 石蕊/酚酞指示剂 | 紫甘蓝汁液 |
| 酸性溶液 | 白醋/柠檬汁 |
| 碱性溶液 | 小苏打水/肥皂水 |

### 制作方法
1. 紫甘蓝切碎，用热水浸泡约10分钟
2. 滤出紫色汁液（即为自制指示剂）
3. 分别滴入白醋（变红）、清水（紫色不变）、小苏打水（变绿/蓝）

### 颜色变化
| 溶液 | 酸碱性 | 紫甘蓝指示剂颜色 |
|------|--------|----------------|
| 白醋/柠檬汁 | 酸性 | 红色/粉红色 |
| 清水 | 中性 | 紫色（不变） |
| 小苏打水/肥皂水 | 碱性 | 绿色/黄色 |

### 科学原理
紫甘蓝中含有花青素，花青素在不同酸碱度溶液中呈现不同颜色——酸性变红、中性紫色、碱性变绿。

### 🚫 居家禁止事项
- 紫甘蓝汁液会染色衣物，操作时注意防护
- 测试液体不可食用
- 实验后及时清洗容器
- 不要使用强酸强碱测试（仅限家中安全物品）

## HOME003 · 铁钉生锈条件探究

### 实验对应
课标实验：金属的锈蚀与防护（铁生锈条件）

### 家用器材
| 材料 | 用途 |
|------|------|
| 铁钉 ×4 | 实验对象 |
| 透明杯 ×4 | 容器 |
| 自来水 | 提供水环境 |
| 凉白开+植物油 | 隔绝空气的水环境 |
| 干燥剂（食品包装内） | 干燥环境 |
| 盐水 | 加速生锈环境 |

### 实验设置
| 组别 | 条件 | 预期结果 |
|------|------|---------|
| 杯1 | 铁钉 + 自来水（敞口） | 生锈（有O₂+H₂O） |
| 杯2 | 铁钉 + 凉白开 + 植物油密封 | 不生锈（有水无O₂） |
| 杯3 | 铁钉 + 干燥剂密封 | 不生锈（有O₂无水） |
| 杯4 | 铁钉 + 盐水（敞口） | 生锈最快（有O₂+H₂O+电解质） |

### 观察周期
3-5天，每天拍照记录铁钉表面变化

### 实验结论
**铁生锈的条件：铁同时与氧气（O₂）和水（H₂O）接触。**
盐水中的铁钉生锈最快——说明电解质加速铁的锈蚀。

### 化学原理
铁锈的主要成分：Fe₂O₃·xH₂O（氧化铁水合物），疏松多孔，无保护作用。

### 🚫 居家禁止事项
- 铁钉尖锐小心划伤
- 盐水组生锈最快，不要用手直接触摸铁锈
- 实验后妥善处理生锈铁钉和水
- 观察期间不要移动杯子影响实验条件

## 居家实验通用安全原则

1. 所有居家实验必须**家长陪同**，不得独自进行
2. 只使用清单中列出的家用安全材料，不自行替换化学品
3. 实验在通风处进行（厨房或阳台）
4. 实验材料和产物均不可食用
5. 实验后及时洗手、清理台面、妥善处理实验废弃物
6. 若出现皮肤刺激或不适，立即停止并用水冲洗

> 以上3个居家实验打通课本知识与生活场景，培养"生活化学"联动思维。核心区别：实验室操作规范 ≠ 居家安全操作。

## HOME004 · 蛋壳与白醋的反应（探究碳酸盐性质）

### 化学方程式
CaCO₃ + 2CH₃COOH → (CH₃COO)₂Ca + H₂O + CO₂↑

### 简易器材
- 鸡蛋壳（去除内膜，洗净晾干）
- 白醋（食醋，含约 5% 醋酸）
- 透明玻璃杯、气球、棉线

### 实验步骤
| 步骤 | 操作 | 现象 |
|------|------|------|
| 1 | 玻璃杯中加入约 30 mL 白醋 | 无色液体 |
| 2 | 投入约 5 g 碎鸡蛋壳 | 立即产生气泡 |
| 3 | 迅速在杯口套上气球，棉线扎紧 | 气球逐渐膨胀 |
| 4 | 静置 10 分钟观察 | 气球不再增大 |

### 标准现象
- 鸡蛋壳表面剧烈产生气泡（CO₂）
- 气球膨胀（收集 CO₂）
- 反应完毕后鸡蛋壳部分溶解

### 探究思考
1. **如何验证气球内是 CO₂？** 将气球内气体导入澄清石灰水，变浑浊即证明。
2. **为何实验室用稀盐酸而居家用白醋？** 醋酸弱酸反应温和、安全；盐酸腐蚀性强不适于居家。
3. **能否用此法检验珍珠/大理石的真伪？** 真珍珠（含 CaCO₃）遇酸冒泡；塑料仿品不反应。

### 🚫 居家禁止事项
- 气球不可过度充气（防止破裂溅出液体）
- 不可同时使用多种清洁剂（如含漂白剂的 84 消毒液 + 酸 → 产生剧毒氯气）
- 实验后液体可倒入下水道，但需大量水冲洗

## HOME005 · 自制简易灭火器（厨房版）

### 化学方程式
Na₂CO₃ + 2CH₃COOH → 2CH₃COONa + H₂O + CO₂↑

### 简易器材
- 空塑料瓶（500 mL 矿泉水瓶）
- 小苏打粉末（NaHCO₃ 或 Na₂CO₃）20 g
- 白醋 200 mL
- 纸巾、棉线

### 实验步骤
| 步骤 | 操作 | 现象 |
|------|------|------|
| 1 | 塑料瓶中加入 20 g 小苏打 | 白色粉末 |
| 2 | 用纸巾包裹 10 g 小苏打做成"延迟包"，棉线拴住 | 延迟反应 |
| 3 | 瓶中加入 100 mL 白醋 | — |
| 4 | 延迟包悬挂在瓶内（不接触白醋），迅速盖紧瓶盖 | — |
| 5 | 倒置塑料瓶 → 包内小苏打接触白醋 | 剧烈产生气泡，瓶内压强增大 |

### ⚠️ 安全警示（重要）
- **此实验模拟灭火器原理，不可对准人或动物喷射**
- 瓶内压强较大，必须使用耐压塑料瓶（不可用玻璃瓶，玻璃碎片危险）
- 实验在户外或浴室进行（易于清理）
- 实验后立即冲洗瓶子和地面

### 探究思考
1. **为何用纸巾包裹小苏打？** 延迟反应时间，便于盖紧瓶盖。
2. **如何验证产物中有 Na₂CO₃？** 取反应后液体 + 稀盐酸，若产生气泡证明含 CO₃²⁻。
3. **CO₂ 灭火原理？** CO₂ 密度比空气大，覆盖可燃物表面，隔绝氧气。

### 🚫 居家禁止事项
- 禁止使用玻璃瓶（炸裂碎片伤人）
- 禁止对准面部喷射
- 禁止大量配制（反应剧烈可能爆瓶）

## HOME006 · 自制火山喷发模型（小苏打 + 白醋）

### 化学方程式
NaHCO₃ + CH₃COOH → CH₃COONa + H₂O + CO₂↑

### 简易器材
- 平底盘（烤盘或大碟子）
- 小塑料杯（150 mL，模拟火山口）
- 面团/黏土（搭建火山外形）
- 小苏打 3 汤匙、白醋 100 mL
- 红色食用色素（模拟岩浆）、洗洁精 1 滴（产生泡沫）

### 实验步骤
| 步骤 | 操作 |
|------|------|
| 1 | 用面团在塑料杯周围堆成锥形火山（杯口为火山口） |
| 2 | 杯中加入 3 汤匙小苏打 + 1 滴洗洁精 + 红色色素 |
| 3 | 迅速倒入 50 mL 白醋 |
| 4 | 观察"火山喷发"：红色泡沫从杯口涌出 |

### 标准现象
- 小苏打与白醋剧烈反应产生 CO₂
- 红色泡沫从"火山口"涌出（CO₂ 推动液体 + 洗洁精产生持久泡沫）

### 探究思考
1. **洗洁精的作用？** 降低液体表面张力，CO₂ 形成持久泡沫（视觉效果增强）。
2. **能否用柠檬汁替代白醋？** 柠檬酸同样为弱酸，可与小苏打反应产生 CO₂。
3. **为何小苏打常用于烘焙？** 受热分解产生 CO₂（2NaHCO₃ → Na₂CO₃ + H₂O + CO₂↑），使面团蓬松。

### 🚫 居家禁止事项
- 不可食用实验产物（含食用色素但仍含化学品）
- 实验在易清理处进行（厨房台面或户外）
- 佩戴围裙防止色素污染衣物

## HOME007 · 维生素 C 的检测（碘伏褪色法）

### 化学原理
维生素C（抗坏血酸 C₆H₈O₆）具有强还原性，可将碘（I₂）还原为碘离子（I⁻），使碘伏溶液褪色。

### 化学方程式（简化）
C₆H₈O₆ + I₂ → C₆H₆O₆ + 2HI

### 简易器材
- 碘伏（药店有售，0.5% I₂ 溶液）
- 柠檬、橙子、苹果、葡萄（不同果汁）
- 淀粉（增强碘的蓝色显示）
- 透明杯子若干、滴管

### 实验步骤
| 步骤 | 操作 | 现象 |
|------|------|------|
| 1 | 4 个杯子各加入 10 mL 水 + 2 滴碘伏 + 少量淀粉 | 蓝色溶液 |
| 2 | 分别挤入 5 mL 不同果汁（柠檬/橙子/苹果/葡萄） | — |
| 3 | 观察褪色时间和程度 | 褪色越快维 C 含量越高 |

### 标准现象
- 柠檬汁：褪色最快（维 C 含量最高）
- 橙汁：快速褪色
- 葡萄汁：缓慢褪色
- 苹果汁：褪色最慢

### 探究思考
1. **维 C 含量排序依据？** 褪色速度 → 单位体积内维 C 浓度排序。
2. **为何维 C 可作抗氧化剂？** 易失去电子（被氧化），保护其他物质免受氧化。
3. **如何定量测定维 C？** 实验室用碘量法滴定（已知浓度 I₂ 滴定维 C 溶液）。

### 🚫 居家禁止事项
- 碘伏外用不可内服
- 果汁必须新鲜（氧化后维 C 减少）
- 实验后液体不可饮用

## HOME008 · 结晶法分离食盐（蒸发结晶 vs 冷却结晶）

### 实验原理
NaCl 溶解度随温度变化小（20℃ 36 g / 100℃ 40 g），适合蒸发结晶。

### 简易器材
- 食盐 50 g、清水 200 mL
- 平底锅（蒸发皿替代品）
- 玻璃棒（筷子替代）
- 过滤网/咖啡滤纸

### 实验步骤
| 步骤 | 操作 | 现象 |
|------|------|------|
| 1 | 50 g 食盐 + 100 mL 热水搅拌溶解 | 透明饱和溶液 |
| 2 | 过滤（如有杂质） | 滤液澄清 |
| 3 | 平底锅小火加热蒸发 | 水分减少 |
| 4 | 当剩余约 1/3 时停止加热 | 析出 NaCl 晶体 |
| 5 | 自然冷却 | 立方体晶体析出 |

### 标准现象
- 蒸发过程中溶液保持饱和，不断有 NaCl 析出
- 最终得到白色立方体 NaCl 晶体

### 探究思考
1. **为何用蒸发法而非冷却法？** NaCl 溶解度受温度影响小，冷却结晶产率低。
2. **如何制备大晶体？** 缓慢蒸发 + 悬挂晶种（提供结晶中心）。
3. **重结晶法提纯：** 粗盐溶解 → 过滤 → 蒸发 → 重结晶，可提高纯度。

### 🚫 居家禁止事项
- 加热时小心烫伤（必须家长陪同）
- 蒸发时不停搅拌防止飞溅
- 实验产物不可食用（可能混入杂质）

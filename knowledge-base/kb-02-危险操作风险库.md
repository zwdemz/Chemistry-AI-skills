# 危险操作风险库

> 覆盖初中化学实验室百余种违规操作的典型案例，每项含风险识别、事故推演、规范操作、应急处理四步解析，配套安全红线口诀。

## DNG001 · 浓硫酸稀释操作错误：将水倒入浓硫酸中

> 📌 关联条目：扩展推演见 DNG020（水入酸事故详细机理）

### ⚠️ 风险识别
稀释浓硫酸时，将水倒入浓硫酸中（而非将浓硫酸注入水中）。

### 💥 事故推演
浓硫酸溶于水放出大量热。水的密度小于浓硫酸，水浮在浓硫酸表面。接触面局部温度骤升使水沸腾，造成液体飞溅。飞溅的浓硫酸可导致皮肤严重化学灼伤、衣物碳化，溅入眼睛可致失明。

### ✅ 规范操作
**将浓硫酸沿容器内壁缓慢注入水中，同时用玻璃棒不断搅拌。**
浓硫酸密度大于水，缓慢注入时沉入水底，搅拌使热量均匀分散。

### 🚑 应急处理
1. 皮肤接触：先用干布轻轻擦拭，再用大量流水冲洗至少15分钟，涂3%-5%碳酸氢钠溶液
2. 眼睛溅入：立即用大量清水冲洗，翻开眼睑，迅速就医
3. 衣物沾到：立即脱去污染衣物

### 🔴 安全红线口诀口诀
**酸入水，沿壁倒，不断搅！**

## DNG002 · 加热试管时试管口朝向自己或他人

> 📌 关联条目：液体加热侧重见本条；满试管加热见 DNG019；固体加热朝人见 DNG030

### ⚠️ 风险识别
加热试管内液体时，试管口朝向自己或他人方向。

### 💥 事故推演
加热过程中液体可能暴沸喷出，管口朝向人可直接导致面部、眼睛化学灼伤或热液烫伤。

### ✅ 规范操作
试管口应朝向无人方向，与桌面呈45°角。加热前应预热试管，液体体积不超过试管容积的1/3。

### 🚑 应急处理
1. 热液烫伤：立即用冷水冲洗降温至少10分钟
2. 化学灼伤：区分酸碱性后对应处理
3. 严重烧伤：覆盖干净纱布，立即就医

### 🔴 安全红线口诀口诀
**管口不对人，预热再集中！**

## DNG003 · 用燃着的酒精灯引燃另一盏酒精灯

> 📌 关联条目：扩展事故推演见 DNG025

### ⚠️ 风险识别
用已经点燃的酒精灯去引燃另一盏酒精灯（对火）。

### 💥 事故推演
倾斜酒精灯时酒精可能从灯口洒出并被引燃，形成流淌火焰，可引燃桌面、衣物和周围可燃物，造成火灾。

### ✅ 规范操作
必须用火柴或打火机点燃酒精灯。熄灭酒精灯必须用灯帽盖灭，盖灭后再揭开灯帽重新盖一次。

### 🚑 应急处理
1. 酒精洒出着火：立即用湿布覆盖灭火（隔绝空气）
2. 身上着火：就地打滚或用灭火毯覆盖
3. 桌面着火：用灭火器（干粉/CO₂）或沙土覆盖

### 🔴 安全红线口诀口诀
**点燃用火柴，禁止灯点灯！**

## DNG004 · 闻气体时直接将鼻子凑到瓶口

### ⚠️ 风险识别
直接凑近试剂瓶口闻气体气味。

### 💥 事故推演
许多化学气体具有刺激性、腐蚀性或毒性（如Cl₂、SO₂、NO₂、NH₃等）。直接闻气体可能导致呼吸道灼伤、中毒、头晕甚至昏迷。

### ✅ 规范操作
**用手在瓶口轻轻扇动，使极少量气体飘入鼻孔（扇闻法）。** 任何时候都不要将鼻子凑近瓶口直接闻。

### 🚑 应急处理
1. 吸入刺激性气体：立即转移到通风处
2. 呼吸困难：保持半卧位，拨打120
3. 昏迷：侧卧防止窒息，等待救援

### 🔴 安全红线口诀口诀
**扇闻不直闻，安全第一位！**

## DNG005 · 实验废液直接倒入水槽

### ⚠️ 风险识别
将化学实验后的废液直接倒入水槽排放。

### 💥 事故推演
化学废液可能含有强酸、强碱、重金属离子（Cu²⁺、Pb²⁺、Ba²⁺）或有机溶剂。直接排放会腐蚀管道，重金属污染水体，有机溶剂引发火灾风险。

### ✅ 规范操作
废液必须倒入指定废液缸，分类收集、统一处理。酸性废液与碱性废液分开，含重金属废液单独收集，有机废液与无机废液分开。

### 🚑 应急处理
1. 废液溅出：根据酸碱性用相应中和剂处理
2. 管道腐蚀泄漏：关闭水源，报告实验室管理员

### 🔴 安全红线口诀口诀
**废液入缸，分类回收！**

## 实验室通用安全守则

1. 进入实验室必须穿实验服，必要时戴护目镜
2. 严禁在实验室内饮食、嬉闹
3. 易燃易爆试剂远离火源，单独存放
4. 实验前预习实验内容，了解所用试剂的安全风险
5. 加热操作必须在通风处或通风橱中进行
6. 实验结束检查水、电、气是否关闭，清洁台面
7. 离开实验室前必须洗手

> 以上5个典型危险操作覆盖加热、试剂稀释、气体处理、废液处置四大类别。化学实验安全的第一原则是"预防为主"。

## DNG006 · 加热固体时试管口朝上倾斜

### ⚠️ 风险识别
加热固体（如高锰酸钾制氧气、碳酸钙分解、碳还原氧化铜）时，试管口朝上倾斜，导致冷凝水倒流。

### 💥 事故推演
1. 加热过程中试剂中的水分或反应生成的水蒸气在试管口冷凝
2. 冷凝水沿试管壁向试管底（加热区）回流
3. 冷凝水接触高温试管底部 → 试管骤冷炸裂
4. 飞溅的玻璃碎片可能割伤手部/面部，热试剂溅出造成灼伤

### ✅ 规范操作
- **试管口略向下倾斜**（倾斜角度约 15°-30°）
- 试管夹夹在距试管口约 1/3 处
- 先均匀预热，再集中加热药品部位
- 加热完毕自然冷却，不可骤冷

### 🚑 应急处理
- 玻璃割伤：清水冲洗伤口 → 碘伏消毒 → 无菌纱布包扎；伤口较深立即就医
- 烫伤：流动冷水冲洗 15-20 分钟 → 涂抹烫伤膏；切勿涂牙膏/酱油

### 🔴 安全红线口诀
**加热固体，试管口向下；倾斜加热，防冷凝水回炸！**

## DNG007 · 加热前未进行气密性检查

### ⚠️ 风险识别
气体发生装置（如制氧气、制 CO₂、制 H₂）使用前未检查气密性，装置漏气导致：
- 有毒气体（CO）泄漏 → 实验室空气污染
- 可燃气体（H₂）泄漏 → 达到爆炸极限遇火源爆炸

### 💥 事故推演
1. 装置接口松动/橡胶塞老化/玻璃管不匹配 → 漏气
2. 气体逸出到实验室空气中
3. 如为 H₂ 或 CH₄ 等可燃气体 → 在空气中累积达到爆炸极限（H₂ 4%-75%）
4. 遇酒精灯明火、电器火花 → **爆燃/爆炸**

### ✅ 规范操作（手握法）
1. 装置组装完毕，导管末端插入水中
2. 双手紧握试管（或锥形瓶外壁）
3. 观察导管口是否有气泡冒出
4. 松开手后，导管内形成一段水柱且不下降
5. 符合上述现象 → 气密性良好

### 🚑 应急处理
- 疑似气体泄漏：立即关闭气源/熄灭酒精灯 → 打开通风设备 → 疏散人员
- 已发生爆燃：立即疏散 → 关闭总电源 → 火警 119

### 🔴 安全红线口诀
**组装完毕先查气密；手握导管见气泡；漏气必拆重装！**

## DNG008 · 用量筒直接加热或配制溶液

### ⚠️ 风险识别
量筒为细长圆柱形玻璃容器，壁薄、底部小、不耐热、不耐骤变，用于加热会导致炸裂。

### 💥 事故推演
1. 量筒壁受热不均 → 玻璃炸裂
2. 量筒内溶液飞溅 → 烫伤/化学灼伤
3. 量筒底部不稳 → 倾倒后溶液洒出
4. 量筒刻度线受热变形 → 后续测量失准

### ✅ 规范操作
- **量筒仅用于量取液体体积，不可加热、不可配制溶液、不可在其中进行化学反应**
- 配制溶液应使用烧杯 + 玻璃棒搅拌
- 加热应使用试管、烧杯、蒸发皿、燃烧匙等耐热容器

### 🚑 应急处理
- 玻璃割伤 + 化学灼伤：先清创 → 大量清水冲洗 → 1% 醋酸（碱灼伤）或 3%-5% 碳酸氢钠（酸灼伤）→ 就医

### 🔴 安全红线口诀
**量筒量取不加热，烧杯配制最合适；薄壁量筒遇热即裂！**

## DNG009 · 点燃氢气前未验纯

### ⚠️ 风险识别
收集的氢气若不纯（混有空气/氧气），点燃时发生剧烈爆炸。

### 💥 事故推演
1. 制氢气 Zn + H₂SO₄ → ZnSO₄ + H₂↑
2. 装置内空气未排尽，氢气与氧气混合
3. H₂ 在空气中的爆炸极限 4%-75%（体积分数）
4. 点燃混合气体 → 瞬间剧烈反应 → **爆鸣/爆炸**（试管炸裂、碎片飞溅）

### ✅ 规范操作（验纯步骤）
1. 收集一试管氢气，用拇指堵住管口
2. 试管口朝下，靠近酒精灯火焰
3. 移开拇指点燃
4. **听声音判断**：
   - "噗"声 → 氢气较纯，可以安全点燃
   - 尖锐爆鸣声 → 氢气不纯，必须重新收集验纯

### 🚑 应急处理
- 试管炸裂割伤：清创 + 碘伏消毒 + 包扎
- 严重炸伤（出血/深度伤口）：立即压迫止血 → 送医

### 🔴 安全红线口诀
**氢气点燃先验纯，"噗"声方可放心用；爆鸣即停莫大意！**

## DNG010 · 一氧化碳尾气未经处理直接排放

### ⚠️ 风险识别
CO 还原氧化铜/氧化铁实验中，未反应的 CO 经导管直接排入空气 → 实验室空气 CO 浓度升高。

### 💥 事故推演
1. CO 是无色无味剧毒气体（与血红蛋白结合能力是 O₂ 的 200-300 倍）
2. 空气中 CO 浓度达 0.02% 即中毒症状；0.1% 以上致命
3. 长时间吸入 → 头晕、恶心、昏迷 → **窒息死亡**
4. CO 与空气混合达爆炸极限（12.5%-74%）遇火源爆炸

### ✅ 规范操作
- **尾气处理三法**：
  1. **气球收集法**（少量 CO）：导管连接气球，收集后集中处理
  2. **燃烧法**（推荐）：尾气出口处用酒精灯点燃 CO → CO₂
  3. **吸收法**：通入 NaOH 溶液（CO 不溶，但可加装二次反应器）
- 实验在通风橱中进行，全程开启通风设备

### 🚑 应急处理
- 轻度中毒（头痛、恶心）：立即转移到通风处 → 吸入新鲜空气 → 必要时吸氧
- 重度中毒（昏迷）：立即拨打 120 → 移至通风处 → 人工呼吸 + 心肺复苏 → 送高压氧舱

### 🔴 安全红线口诀
**CO 尾气必处理；气球收集或点燃；通风橱内方可行！**

## DNG011 · 钾、钠等活泼金属直接投入水中

### ⚠️ 风险识别
钾、钠、钙等极活泼金属与水剧烈反应产生 H₂ + 大量热，可能引发燃烧/爆炸。

### 💥 事故推演
1. 2Na + 2H₂O → 2NaOH + H₂↑（剧烈放热）
2. 钠熔成小球在水面游动，发出嘶嘶声
3. 反应放热使 H₂ 达到着火点 → 燃烧
4. H₂ 与空气混合达爆炸极限 → **爆炸**
5. 钾比钠更活泼，常温下自燃

### ✅ 规范操作
- 初中阶段**严禁将钾、钠投入水中演示**
- 钾、钠必须保存在**煤油或石蜡油**中（隔绝空气和水）
- 用镊子取用，切勿用手接触（汗液含水分）
- 实验剩余的金属碎屑用乙醇缓慢处理（生成乙醇钠 + H₂）

### 🚑 应急处理
- 金属钠引燃：**严禁用水灭火**！使用干燥沙土或干粉灭火器覆盖
- 碱液灼伤：大量清水冲洗 → 2% 硼酸溶液冲洗 → 就医

### 🔴 安全红线口诀
**钾钠入水必爆燃；煤油保存是王道；遇火禁水用沙盖！**

## DNG012 · 药品过量取用（超量反应）

### ⚠️ 风险识别
"节约用药"原则被违反，反应物过量导致：
- 反应过于剧烈 → 液体飞溅
- 气体产生过快 → 装置超压爆炸
- 浪费试剂 + 增加废液处理难度

### 💥 事故推演（举例：大理石 + 过量稀盐酸）
1. 大理石（CaCO₃）与稀盐酸反应：CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑
2. 盐酸过量 → CO₂ 产生速率过快
3. 气体压强超过橡胶塞承受极限 → **塞子弹出/容器炸裂**
4. 酸液飞溅 → 化学灼伤

### ✅ 规范操作
- **三大取用原则**：
  1. 液体取 1-2 mL（最多不超过试管容积的 1/3）
  2. 固体盖满试管底部
  3. 剩余药品不能放回原瓶（放入指定容器）
- 制气体时使用**长颈漏斗 + 多孔隔板**（可控制反应的发生和停止）

### 🚑 应急处理
- 酸液溅入眼睛：立即用大量清水冲洗 15 分钟（眼睑翻开）→ 就医
- 酸液溅到皮肤：立即清水冲洗 → 3%-5% NaHCO₃ 溶液冲洗

### 🔴 安全红线口诀
**液体一两毫升够，固体盖满试管底；节约用药是美德！**

## DNG013 · 用嘴品尝任何化学试剂

### ⚠️ 风险识别
违反"不可尝、不可闻、不可触"的基本原则，试剂入口即中毒。

### 💥 事故推演
1. 误以为"食盐"的 NaCl → 可能是 NaNO₂（亚硝酸钠，0.2 g 致死）
2. 误以为"白糖"的蔗糖 → 可能是葡萄糖/蔗糖混合物（含杂质）
3. 试剂入口 → 急性中毒 → 呕吐/抽搐/休克

### ✅ 规范操作
- **实验室"三不"原则**：
  1. 不要尝任何试剂的味道
  2. 不要直接闻气体（应"扇闻"）
  3. 不要用手直接接触药品
- 如需鉴别 NaCl 和 Na₂CO₃：用稀盐酸（CO₃²⁻ + 2H⁺ → H₂O + CO₂↑）

### 🚑 应急处理
- 误食：立即饮水稀释 → 用手指刺激咽喉催吐（腐蚀性物质禁止催吐）→ 携带试剂标签送医
- 误食强酸/强碱：**禁止催吐**（腐蚀食道二次损伤）→ 立即饮用大量清水/蛋清 → 120

### 🔴 安全红线口诀
**试剂入口即中毒；鉴别必用化学法；误食强酸禁催吐！**

## DNG014 · 直接俯视/仰视读取量筒刻度

### ⚠️ 风险识别
读数时视线与液面不水平，导致取量错误，后续反应浓度/质量分数失准。

### 💥 事故推演
1. 仰视读数 → 实际取量偏小（视线低 → 弯月面看似偏低，实际刻度大于实际液体）
   - 等等，仰视读数 → 视线向上看 → 看到的刻度低于实际液面 → 读数偏小 → 实际取量**偏多**
2. 俯视读数 → 实际取量偏少
3. 浓度偏差 → 反应异常（如中和反应未达终点/已过量）

### ✅ 规范操作
- 视线与液体**凹液面最低处**保持水平
- 量筒平放于实验台，不可手持读数
- 读数到最小刻度后一位（估读）

### 🚑 应急处理
- 无需应急（误差问题），但需重做实验
- 浓度偏差较大需记录在实验报告中

### 🔴 安全红线口诀
**视线与凹液面水平；仰视偏多俯视少；配液务必精确读！**

## DNG015 · 酒精灯内酒精过满或不足

### ⚠️ 风险识别
- 酒精体积 > 灯体容积 2/3 → 内部压强过大 → 点燃时酒精蒸气喷出 → 火焰失控
- 酒精体积 < 灯体容积 1/4 → 灯芯露出液面 → 点燃时空气-酒精混合气爆炸

### 💥 事故推演
1. 过满：灯内酒精蒸气压过高 → 移动时液体溢出 → 遇明火 → 整灯着火
2. 过少：灯芯暴露 + 灯内空气-酒精混合 → 点燃 → **灯内爆燃**（玻璃碎片飞溅）

### ✅ 规范操作
- 酒精灯内酒精量保持在**灯体容积的 1/4 至 2/3 之间**
- 添加酒精必须**用漏斗**，熄灭后冷却再添加
- 严禁燃着时添加酒精（爆燃）

### 🚑 应急处理
- 酒精洒出着火：用湿抹布覆盖灭火（隔绝空气）→ 严禁用水（酒精比水轻，浮在水面继续燃烧）
- 严重烧伤：大量冷水冲洗 → 干净布覆盖 → 立即送医

### 🔴 安全红线口诀
**酒精不超三分二；不少于四分之一；燃时禁止添酒精！**

## DNG016 · 实验室饮食或存放食物

### ⚠️ 风险识别
实验室内饮食、存放食物 → 食物被试剂污染 → 食用后中毒。

- 误食一般试剂：大量饮水稀释 → 携带标签就医

### 🔴 安全红线口诀
**实验室里禁饮食；洗手方能进餐去；食物严禁带进来！**

## DNG017 · 用手直接接触药品

### ⚠️ 风险识别
手上汗液 + 皮肤油脂与试剂反应，腐蚀性试剂灼伤皮肤，重金属经皮肤吸收中毒。

### 💥 事故推演
1. 接触浓硫酸 → 脱水炭化皮肤 → 黑色碳化层 + 剧痛
2. 接触苯酚（C₆H₅OH）→ 渗透性强 → 局部麻醉（不痛）→ 坏死
3. 接触 AgNO₃ → 皮肤蛋白沉淀 → 黑色斑点（持久）
4. 接触 NaOH → 皂化皮肤脂肪 → 滑腻感 → 深度灼伤

### ✅ 规范操作
- **任何药品都不能用手接触**
- 固体用药匙或镊子取用
- 液体用滴管或倾倒法
- 必要时戴橡胶手套（特别是浓酸、浓碱操作）
- 实验后立即洗手

### 🚑 应急处理
- 酸灼伤：大量清水冲洗 → 3%-5% NaHCO₃ 溶液冲洗 → 就医
- 碱灼伤：大量清水冲洗 → 涂抹硼酸溶液（弱酸）→ 就医
- 酚灼伤：大量酒精冲洗（酚溶于酒精）→ 就医

### 🔴 安全红线口诀
**药匙镊子取固体；滴管倾倒取液体；手套护手才安全！**

## DNG018 · 酒精灯打翻着火用水扑救

### ⚠️ 风险识别
酒精灯打翻 → 桌面酒精着火 → 错误用水扑救 → 酒精浮在水面上继续燃烧，火势蔓延。

### 💥 事故推演
1. 酒精灯倾倒 → 桌面酒精燃烧（蓝色火焰）
2. 学生惊慌用水泼 → 酒精比水轻（密度 0.8 g/cm³）→ 酒精浮在水面
3. 流动的水带着燃烧的酒精蔓延 → **火势扩大**
4. 引燃周围可燃物（书本、纸张、衣物）→ 重大事故

### ✅ 规范操作
- 酒精着火：**立即用湿抹布覆盖**（隔绝空气，灭火）
- 大面积着火：使用**干粉灭火器**或**沙土**
- **严禁用水扑救酒精火**
- 衣服着火：就地打滚或用湿布包裹（切勿奔跑助燃）

### 🚑 应急处理
- 轻度烧伤：冷水冲洗 15-20 分钟 → 涂抹烫伤膏
- 中度烧伤：冷水冲洗 → 干净布覆盖 → 立即送医
- 重大火情：疏散人员 → 拨打 119

### 🔴 安全红线口诀
**酒精着火湿布盖；切勿用水助蔓延；就地打滚灭衣火！**

## DNG019 · 给盛满液体的试管直接加热

> 📌 关联条目：满试管液体加热侧重见本条；液体一般加热见 DNG002；固体加热朝人见 DNG030

### ⚠️ 风险识别
试管液体超过 1/3 容积，加热时沸腾飞溅；试管口对人导致烫伤。

### 💥 事故推演
1. 试管液体过满 → 加热剧烈沸腾 → 液体从管口喷出
2. 试管口朝向自己或同学 → 沸腾液体溅入眼睛/皮肤 → **烫伤/化学灼伤**
3. 若为酸性/碱性液体 → 双重伤害

### ✅ 规范操作

- 试管口**朝向无人区**（45°倾斜，自己与他人面部方向禁止）

- 加热时不断振荡（不可对人）

### 🚑 应急处理
- 沸水溅入眼睛：立即清水冲洗 15 分钟 → 眼科急诊
- 酸/碱液溅入眼睛：立即清水冲洗 15 分钟（眼睑翻开）→ 急诊

### 🔴 安全红线口诀
**液体不超三分一；管口朝向无人区；夹持振荡防飞溅！**

## DNG020 · 稀释浓硫酸时将水倒入浓硫酸

> 📌 关联条目：基础条目见 DNG001（浓硫酸稀释操作错误）；本条为水入酸事故详细机理推演

### ⚠️ 风险识别
（深化 DNG001 的扩展说明）**水入酸**错误顺序导致剧烈放热、酸液飞溅。

### 💥 事故推演
1. 浓硫酸密度 1.84 g/cm³ > 水密度 1.0 g/cm³
2. 水浮在浓硫酸上方
3. 溶解放热集中在两液接触面（水层底部）
4. 局部水瞬间沸腾 → **酸液飞溅**（酸滴射向四周）
5. 飞溅的酸滴腐蚀皮肤、眼睛、衣物

### ✅ 规范操作
- **正确顺序：浓硫酸沿烧杯内壁缓慢倒入水中**，边倒边用玻璃棒搅拌散热
- "酸入水、沿器壁、慢慢倒、不断搅"
- 稀释在烧杯中进行（不可用量筒）
- 操作人员穿戴护目镜 + 实验服 + 手套

### 🚑 应急处理
- 皮肤溅酸：立即大量清水冲洗 15 分钟 → 3%-5% NaHCO₃ 溶液湿敷 → 急诊
- 眼睛溅酸：眼睑翻开清水冲洗 15 分钟（必须）→ 立即眼科急诊
- 衣物腐蚀：立即脱除（防止皮肤吸收）

### 🔴 安全红线口诀
**稀释酸入水；沿壁慢慢倒；玻璃棒搅拌散热！**

## DNG021 · 实验室加热液体未加沸石（防暴沸）

### ⚠️ 风险识别
加热液体（如蒸馏水、配制溶液）时未加沸石或碎瓷片 → 形成过热液体 → 突然剧烈沸腾（暴沸）。

### 💥 事故推演
1. 加热纯液体时，无汽化中心（无凝结核）
2. 液体温度超过沸点仍不沸腾（过热状态）
3. 任何扰动（振动、加沸石）→ 瞬间大量汽化中心 → **暴沸**
4. 液体剧烈喷出 → 烫伤

### ✅ 规范操作
- 加热液体前加入 1-2 粒沸石（或碎瓷片）
- 沸石不可反复使用（已湿润的沸石无汽化中心）
- 中途补加沸石必须**先停止加热**，液体冷却后再加
- 蒸馏操作必须持续监测温度

### 🚑 应急处理
- 暴沸溅出：冷水冲洗 → 烫伤处理
- 大面积溅出：立即疏散 → 关闭热源 → 通风

### 🔴 安全红线口诀
**加热液体加沸石；冷却补加莫大意；防止暴沸最关键！**

## DNG022 · 浓酸浓碱试剂瓶标签未盖好或朝下

### ⚠️ 风险识别
倾倒浓酸/浓碱时标签朝下 → 残留液腐蚀标签 → 试剂信息丢失 → 误用。

### 💥 事故推演
1. 倾倒时瓶口残留液腐蚀标签（浓硫酸脱水使标签变黑，浓碱使标签水解）
2. 标签脱落 → 瓶内试剂无法识别
3. 误将 NaOH 当作 NaCl → 与酸反应剧烈放热飞溅
4. 误将 H₂SO₄ 当作 H₂O → 配制溶液顺序错误

### ✅ 规范操作
- **倾倒液体时标签朝向手心**
- 瓶塞倒放在桌面上（不可正放污染）
- 取用完毕立即盖紧瓶塞
- 试剂瓶放回原处，标签朝外便于取用

### 🚑 应急处理
- 标签丢失：立即标识"待鉴定"，禁止使用
- 试剂未知：取样做性质实验鉴定（测 pH、加酸/碱观察反应）

### 🔴 安全红线口诀
**倾倒标签朝手心；瓶塞倒放防污染；信息丢失即鉴定！**

## DNG023 · 用排水法收集氧气后先熄灭酒精灯

### ⚠️ 风险识别
高锰酸钾制氧气实验中，停止实验时先熄灭酒精灯而不先将导管移出水面 → 水倒吸。

### ✅ 规范操作
- **正确顺序：先把导管移出水面，再熄灭酒精灯**
- 实验完毕"先撤后熄"
- 自然冷却至室温后再拆卸装置

### 🚑 应急处理
- 试管炸裂：清理碎片 → 高锰酸钾溶液污染用清水冲洗
- 割伤：清创 + 碘伏 + 包扎

### 🔴 安全红线口诀
**制氧停止先撤管，再熄酒精灯最安；倒吸炸裂记心间！**

## DNG024 · 过滤时液面高于滤纸边缘

### ⚠️ 风险识别
过滤操作中，倒入的悬浊液液面高于滤纸边缘 → 悬浊液沿滤纸与漏斗壁之间的缝隙流下 → 滤液仍含杂质。

### 💥 事故推演
1. 液面超过滤纸边缘
2. 液体从滤纸外侧（非过滤面）流下
3. 未经过滤直接进入承接烧杯
4. 滤液浑浊 → 需重新过滤（耗时）
5. 严重时滤纸被冲破 → 整个实验失败

### ✅ 规范操作
- 滤纸边缘应**低于漏斗口**（约 0.5 cm）
- 倾倒液体时**液面低于滤纸边缘**
- 使用玻璃棒引流（三靠：玻璃棒靠三层滤纸处、玻璃棒下端靠烧杯内壁、漏斗下端靠烧杯内壁）
- 一次性不可倒入过多

### 🚑 应急处理
- 无需应急（实验失败问题），需重新过滤
- 若滤液污染：弃用重新配制

### 🔴 安全红线口诀
**液面低于滤纸边；玻璃棒引三靠稳；一次少量分次加！**

## DNG025 · 用燃着的酒精灯引燃另一只酒精灯

## DNG028 · 加热易暴沸液体（如食用油）未控温

### ⚠️ 风险识别
加热食用油、浓硫酸等粘稠液体，温度超过沸点后突然暴沸 → 高温液体飞溅。

### 💥 事故推演
1. 食用油沸点约 200-300℃，加热过程中无明显沸腾（粘度大）
2. 局部过热（超过沸点仍未沸腾）
3. 任何扰动 → 瞬间大量汽化 → **暴沸**
4. 高温油雾飞溅 → 二度烧伤

- 加热粘稠液体必须**加沸石或碎瓷片**
- 控制加热温度（使用温度计）
- 缓慢升温，避免局部过热
- 操作人员穿戴护目镜、隔热手套、围裙

### 🚑 应急处理
- 高温油溅出：立即冷水冲洗 15-20 分钟 → 涂烫伤膏
- 大面积烫伤：冷水冲洗 → 干净布覆盖 → 立即送医
- 油烟吸入：转移到通风处 → 必要时吸氧

### 🔴 安全红线口诀
**粘稠液体加沸石；缓慢升温防爆沸；护目手套必穿戴！**

## DNG029 · 强氧化剂与还原剂混合存放（如 KClO₃ 与红磷/硫）

### ⚠️ 风险识别
强氧化剂（KClO₃、KMnO₄、KNO₃）与还原剂（红磷、硫磺、木炭）混合存放 → 摩擦/受热即爆炸。

### 💥 事故推演
1. KClO₃ 与红磷混合 → 受到摩擦或撞击 → 剧烈反应
2. KClO₃ + 3P → 3KCl + P₃ → 爆炸
3. 实验室存储不当 → 玻璃瓶碰撞、试剂瓶跌落 → **爆炸**
4. 历史事故：氯酸钾 + 红磷 = " SOUND" 噪声剂，摩擦即爆

### ✅ 规范操作
- 强氧化剂与还原剂**分柜存放**（隔离储存）
- 实验操作时**单独取用**，不可同时放在桌面
- 操作台清洁，无残留物
- 实验剩余物分别回收，**严禁混合倾倒**

### 🚑 应急处理
- 已混合尚未反应：立即用大量水冲洗分解（少量）
- 已发生爆炸：立即疏散 → 拨打 119 + 120
- 烧伤：冷水冲洗 → 干净布覆盖 → 送医

### 🔴 安全红线口诀
**氧化还原分柜存；同台操作最危险；混合即爆莫混合！**

## DNG030 · 试管加热时朝向有人方向（自己或同学）

> 📌 关联条目：固体加热朝人侧重见本条；液体加热朝人见 DNG002；满试管加热见 DNG019

### ⚠️ 风险识别
加热试管时，试管口朝向自己或同学面部 → 沸腾液体溅出 → 烫伤眼睛/面部。

### 💥 事故推演
1. 试管口对人加热
2. 液体暴沸或沸腾 → 液体从管口喷出
3. 喷向操作者或同学 → **烫伤眼睛（失明风险）/面部/颈部**
4. 若为酸碱液 → **双重化学灼伤**

### ✅ 规范操作
- 加热时试管口**朝向无人区**（空旷方向）
- 试管与桌面成 45° 角
- 试管夹夹在距试管口 1/3 处
- 先均匀预热（移动火焰），再集中加热

### 🚑 应急处理
- 沸水溅入眼睛：立即清水冲洗 15 分钟（眼睑翻开）→ 眼科急诊
- 酸碱液溅入眼睛：立即清水冲洗 15-30 分钟 → 急诊
- 面部烫伤：冷水冲洗 → 干净布覆盖 → 送医

### 🔴 安全红线口诀
**试管加热朝无人；45°角倾斜持；夹在距口三分一！**

## DNG031 · 实验结束后未关闭水电气（水龙头、电源、燃气）

### ⚠️ 风险识别
实验结束离开未检查水龙头、电源、燃气 → 漏水/火灾/中毒。

### 💥 事故推演
1. 水龙头未关 → 长时间流水 → 漫溢损坏设备 → 楼下漏水
2. 电热设备未拔电源 → 长时间加热 → 过热起火
3. 燃气（如本生灯）未关 → 燃气泄漏 → 中毒 + 爆炸
4. 夜间无人发现 → 重大事故

### ✅ 规范操作
- 实验结束"三检查"：
  1. 水龙头全关
  2. 电热设备拔电源（不只是关开关）
  3. 燃气阀门关闭
- 清洁台面，试剂归位
- 最后离开者再次检查

### 🚑 应急处理
- 漏水：立即关水龙头 → 清理积水 → 报修
- 火灾：切断电源 → 灭火器 → 拨打 119
- 燃气泄漏：**严禁开关电器（火花引爆）** → 关闭总阀 → 通风 → 撤离 → 报警

### 🔴 安全红线口诀
**实验结束三检查；水电气阀全关闭；最后离场再确认！**

## 四、路径使用规范

### 4.1 引用格式
```
[kb-01#EXP001]  → 课标必考实验库 / EXP001 高锰酸钾制氧气
[kb-02#DNG001]  → 危险操作风险库 / DNG001 浓硫酸稀释
[kb-06#第十单元] → 考点速查 / 第十单元 酸和碱
```

### 4.2 检索优先级
1. **具体实验**：kb-01 → kb-03（变量对照）
2. **安全规范**：kb-02 → kb-05（闯关题）→ kb-04（居家安全）
3. **知识点查询**：kb-06（方程式/考点速查）
4. **生活拓展**：kb-04（居家实验）

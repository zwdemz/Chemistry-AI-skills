# 化学方程式速查表

> 人教版初中化学核心方程式（60+ 条），按反应类型与单元双重编排。

---

## 一、化合反应（多变一）

| 方程式 | 条件 | 现象 | 单元 |
|--------|------|------|:----:|
| 4P + 5O₂ →点燃→ 2P₂O₅ | 点燃 | 大量白烟，放热 | 二 |
| C + O₂ →点燃→ CO₂ | 点燃（充分）| 发白光 | 二/六 |
| 2C + O₂ →点燃→ 2CO | 点燃（不充分）| — | 六 |
| S + O₂ →点燃→ SO₂ | 点燃 | 蓝紫色火焰，刺激味 | 二 |
| 3Fe + 2O₂ →点燃→ Fe₃O₄ | 点燃 | 火星四射，黑色固体 | 二/八 |
| 2H₂ + O₂ →点燃→ 2H₂O | 点燃 | 淡蓝色火焰 | 四/七 |
| 2CO + O₂ →点燃→ 2CO₂ | 点燃 | 蓝色火焰 | 六 |
| CO₂ + H₂O → H₂CO₃ | 常温 | 石蕊变红 | 六/十 |

## 二、分解反应（一变多）

| 方程式 | 条件 | 现象 | 单元 |
|--------|------|------|:----:|
| 2H₂O →通电→ 2H₂↑ + O₂↑ | 通直流电 | 正氧负氢，体积比 1:2 | 四 |
| 2KMnO₄ →△→ K₂MnO₄ + MnO₂ + O₂↑ | 加热 | 紫黑色变黑 | 二 |
| 2KClO₃ →MnO₂、△→ 2KCl + 3O₂↑ | 加热+催化剂 | 持续分解 | 二 |
| 2H₂O₂ →MnO₂→ 2H₂O + O₂↑ | 催化剂 | 带火星木条复燃 | 二 |
| CaCO₃ →高温→ CaO + CO₂↑ | 高温 | 工业制生石灰 | 六/十一 |
| H₂CO₃ → H₂O + CO₂↑ | 常温 | 碳酸不稳定 | 六 |
| CuSO₄·5H₂O →△→ CuSO₄ + 5H₂O | 加热 | 蓝色晶体变白色粉末 | 九 |

## 三、置换反应（单质+化合物→新单质+新化合物）

### 金属 + 酸 → 盐 + H₂↑
| 方程式 | 现象 | 单元 |
|--------|------|:----:|
| Mg + 2HCl → MgCl₂ + H₂↑ | 剧烈气泡 | 八/十 |
| 2Al + 6HCl → 2AlCl₃ + 3H₂↑ | 剧烈气泡 | 八/十 |
| Zn + H₂SO₄ → ZnSO₄ + H₂↑ | 实验室制氢气 | 八/十 |
| Fe + 2HCl → FeCl₂ + H₂↑ | 缓慢气泡，溶液变浅绿 | 八/十 |
| Fe + H₂SO₄ → FeSO₄ + H₂↑ | 缓慢气泡 | 八/十 |

### 金属 + 盐 → 新金属 + 新盐
| 方程式 | 现象 | 单元 |
|--------|------|:----:|
| Fe + CuSO₄ → FeSO₄ + Cu | 铁丝附红色固体，溶液蓝→浅绿 | 八 |
| Cu + 2AgNO₃ → Cu(NO₃)₂ + 2Ag | 铜附银白固体，溶液变蓝 | 八 |
| 2Al + 3CuSO₄ → Al₂(SO₄)₃ + 3Cu | 铝附红色固体 | 八 |

### 碳/氢气还原金属氧化物
| 方程式 | 现象 | 单元 |
|--------|------|:----:|
| C + 2CuO →高温→ 2Cu + CO₂↑ | 黑色变红色 | 六 |
| CO + CuO →△→ Cu + CO₂ | 黑色变红色 | 六 |
| 3CO + Fe₂O₃ →高温→ 2Fe + 3CO₂ | 红色变黑色（高炉炼铁）| 八 |

## 四、复分解反应（酸碱盐）

### 酸 + 碱（中和反应）
| 方程式 | 现象 | 单元 |
|--------|------|:----:|
| HCl + NaOH → NaCl + H₂O | 无明显现象，放热 | 十 |
| H₂SO₄ + 2NaOH → Na₂SO₄ + 2H₂O | 无明显现象 | 十 |
| HCl + Ca(OH)₂ → CaCl₂ + 2H₂O | 无明显现象 | 十 |

### 酸 + 金属氧化物 → 盐 + 水
| 方程式 | 现象 | 单元 |
|--------|------|:----:|
| Fe₂O₃ + 6HCl → 2FeCl₃ + 3H₂O | 除铁锈，溶液变黄 | 八/十 |
| Fe₂O₃ + 3H₂SO₄ → Fe₂(SO₄)₃ + 3H₂O | 除铁锈 | 八/十 |
| CuO + H₂SO₄ → CuSO₄ + H₂O | 黑色溶解，溶液变蓝 | 十 |

### 酸 + 盐 → 新酸 + 新盐
| 方程式 | 现象 | 单元 |
|--------|------|:----:|
| CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑ | 实验室制 CO₂ | 六/十 |
| Na₂CO₃ + 2HCl → 2NaCl + H₂O + CO₂↑ | 产生气泡 | 十/十一 |
| K₂CO₃ + 2HCl → 2KCl + H₂O + CO₂↑ | 产生气泡 | 十一 |
| HCl + AgNO₃ → AgCl↓ + HNO₃ | 白色沉淀（不溶硝酸）| 十/十一 |
| H₂SO₄ + BaCl₂ → BaSO₄↓ + 2HCl | 白色沉淀（不溶硝酸）| 十/十一 |

### 碱 + 非金属氧化物 → 盐 + 水
| 方程式 | 现象 | 单元 |
|--------|------|:----:|
| 2NaOH + CO₂ → Na₂CO₃ + H₂O | 吸收 CO₂（防变质）| 十 |
| 2NaOH + SO₂ → Na₂SO₃ + H₂O | 处理 SO₂ 废气 | 十 |
| Ca(OH)₂ + CO₂ → CaCO₃↓ + H₂O | 石灰水变浑浊（检验 CO₂）| 六/十 |

### 碱 + 盐 → 新碱 + 新盐
| 方程式 | 现象 | 单元 |
|--------|------|:----:|
| CuSO₄ + 2NaOH → Cu(OH)₂↓ + Na₂SO₄ | 蓝色沉淀 | 十 |
| FeCl₃ + 3NaOH → Fe(OH)₃↓ + 3NaCl | 红褐色沉淀 | 十 |
| Na₂CO₃ + Ca(OH)₂ → CaCO₃↓ + 2NaOH | 工业制烧碱 | 九/十/十一 |

### 盐 + 盐 → 新盐 + 新盐
| 方程式 | 现象 | 单元 |
|--------|------|:----:|
| NaCl + AgNO₃ → AgCl↓ + NaNO₃ | 白色沉淀 | 十一 |
| Na₂SO₄ + Ba(NO₃)₂ → BaSO₄↓ + 2NaNO₃ | 白色沉淀 | 十一 |
| CaCl₂ + Na₂CO₃ → CaCO₃↓ + 2NaCl | 白色沉淀 | 九 |

## 七、记忆口诀

| 类型 | 口诀 |
|------|------|
| 反应类型 | 化合多变一，分解一变多，置换单换单，复分解双交换 |
| 金属活动性 | K Ca Na Mg Al Zn Fe Sn Pb (H) Cu Hg Ag Pt Au |
| 复分解条件 | 生成沉淀、气体或水即可 |
| Cl⁻ 检验 | 加 AgNO₃ + 稀硝酸，白色沉淀 |
| SO₄²⁻ 检验 | 加 BaCl₂ + 稀硝酸，白色沉淀 |
| CO₃²⁻ 检验 | 加稀盐酸，气体通入澄清石灰水变浑浊 |
| NH₄⁺ 检验 | 加碱加热，气体使湿润红色石蕊试纸变蓝 |

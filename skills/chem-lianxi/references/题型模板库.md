# 题型模板库

> 初中化学 4 大题型 17 种变体的出题/答题模板，配合 chem-lianxi skill 使用。

---

## 一、选择题（4 种变体）

### 模板 1.1 单选题（基础概念）
**结构**：题干（一句概念/现象）→ 4 选项（1 正 3 错）
**答题口诀**：排除法，最确定的先选，剩下对比
**示例**（kb-05#Q005）：酒精灯使用绝对禁止？A 用灯帽盖灭 B 燃着的酒精灯点另一盏 C 熄灭后再盖 D 加酒精时先熄灭 → **B**

### 模板 1.2 单选题（方程式判断）
**结构**：题干给出 4 个方程式，选"正确/错误"的
**判断要点**：①化学式正确 ②配平 ③条件标注 ④↑↓ 符号
**易错**：Fe 与酸/盐生成 +2 价（不是 +3）

### 模板 1.3 单选题（实验现象描述）
**结构**：某实验产生的现象是？
**答题原则**：①只描述感官能观察到的（颜色/状态/光/热）②不写结论（"说明是 O₂"不是现象）
**示例**：铁丝在 O₂ 中燃烧现象 → "剧烈燃烧，火星四射，生成黑色固体"（不能写"生成 Fe₃O₄"）

### 模板 1.4 多选题
**结构**：4 选项多选，正确答案 2-4 个
**策略**：逐项判断，不能用排除法（至少有 2 正）

## 二、填空与简答题（5 种变体）

### 模板 2.1 方程式填空
**结构**：给出反应物，填写产物或配平
**口诀**：写化学式 → 配平 → 注条件 → 标 ↑↓
**示例**：__ + __HCl → __ + __H₂↑（填 Fe + 2HCl → FeCl₂ + H₂↑）

### 模板 2.2 实验现象填空
**结构**：某操作后观察到__？
**答题要点**：颜色/状态/光/热/气味变化，不写"说明"
**示例**：硫在 O₂ 中燃烧 → "蓝紫色火焰，放出刺激性气味气体"

### 模板 2.3 原因分析
**结构**：为什么__操作？（或：不这样操作会怎样）
**答题模板**：原因 + 不这样做的后果
**示例**：为什么试管口略向下倾斜加热固体？→ "防止冷凝水（或释放的水）倒流至灼热试管底导致炸裂"

### 模板 2.4 实验设计
**结构**：设计实验验证__猜想
**答题五要素**：①假设 ②变量控制 ③操作步骤 ④预期现象 ⑤结论
**示例**：验证铁生锈需要 O₂ → "取 2 支试管各放铁钉，1 支加水煮沸+油封（无 O₂），1 支敞口加水（有 O₂），观察 3 天后前者不生锈后者生锈，证明铁生锈需要 O₂"

### 模板 2.5 物质鉴别/推断
**结构**：A、B、C 三瓶无色溶液分别是__，如何鉴别？
**答题方法**：选试剂 → 分组（产生不同现象）→ 一一对应
**示例**：鉴别 HCl、NaOH、NaCl → 各取少量加紫色石蕊：变红是 HCl，变蓝是 NaOH，紫色不变是 NaCl

## 三、实验探究题（4 种变体）

### 模板 3.1 控制变量探究
**结构**：探究__对__的影响
**答题框架**：
1. 提出问题
2. 作出假设
3. 设计实验（控制变量 + 自变量 + 因变量）
4. 进行实验
5. 收集数据（表格）
6. 得出结论
7. 反思评价

### 模板 3.2 气体制备综合
**结构**：选择装置制取__气体
**答题要点**：
- 发生装置：固固加热 vs 固液不加热
- 收集装置：排水/向上排空气/向下排空气
- 验满/检验方法
- 除杂/干燥方法

### 模板 3.3 物质组成探究
**结构**：通过实验探究某物质组成
**示例**：电解水探究水的组成 → 测得正极 O₂ 负极 H₂，体积比 1:2 → 结论：水由 H、O 组成

### 模板 3.4 误差分析
**结构**：某实验测得结果偏大/偏小，原因是？
**答题方法**：找出操作偏差 → 判断对结果影响方向
**示例**：量筒俯视读数量取水 → 俯视使读数偏大但实际水量偏少 → 配制溶液质量分数**偏小**

## 四、计算题（4 种变体）

### 模板 4.1 化学方程式计算（六步法）
**六步**：设 → 写 → 标 → 列 → 求 → 答
- 设：设未知数为 x
- 写：写出配平方程式
- 标：标已知量和未知数对应物质的相对分子质量×计量数
- 列：列比例式
- 求：解方程
- 答：作答

**示例**：6.5 g Zn 与足量稀 H₂SO₄ 反应，生成 H₂ 多少克？
- 设：生成 H₂ 质量为 x
- 写：Zn + H₂SO₄ → ZnSO₄ + H₂↑
- 标：65 / 2 = 6.5 / x
- 列：x = 6.5 × 2 / 65 = 0.2
- 答：生成 H₂ 0.2 g

### 模板 4.2 溶质质量分数计算
**公式**：ω = m溶质 / m溶液 × 100%
**稀释公式**：m₁ω₁ = m₂ω₂
**示例**：50 g 6% NaCl 溶液含水多少？→ m(NaCl) = 50×6% = 3 g；m(H₂O) = 50 - 3 = 47 g

### 模板 4.3 溶解度相关计算
**饱和溶液**：S = 100 × m溶质 / m溶剂
**示例**：20℃ 时 36 g NaCl 溶于 100 g 水恰好饱和，则 S(NaCl, 20℃) = 36 g

### 模板 4.4 综合计算（含杂质/产率）
**结构**：样品含杂质，计算纯度/产率
**要点**：①方程式按纯物质计算 ②杂质不参与反应 ③纯度 = 纯物质质量 / 样品总质量 × 100%

## 五、答题规范通用要点

1. **化学式书写规范**：正价在前负价在后，交叉化简（Ca²⁺和NO₃⁻ → Ca(NO₃)₂）
2. **方程式"四查"**：查化学式、查配平、查条件、查状态符号
3. **现象描述"四不"**：不写结论、不写"可能"、不写未观察到的、不写"说明是"
4. **答题用语规范**：用化学术语（"置换反应"不是"换位反应"）
5. **单位**：计算题必须带单位（g、mL、℃、%）

- 化学式与化合价

## 二、题型 1：一般方程式计算（六步法）

### 解题六步
1. **设**：设未知数为 x
2. **写**：写出配平的化学方程式
3. **标**：标已知量和未知数对应物质的"相对分子质量 × 计量数"
4. **列**：列比例式
5. **求**：解方程
6. **答**：作答（带单位）

### 例题 1（基础）
**题目**：6.5 g 锌与足量稀硫酸完全反应，生成氢气多少克？

**解答**：
- 设生成 H₂ 质量为 x
- Zn + H₂SO₄ → ZnSO₄ + H₂↑
- 65 / 2 = 6.5 / x
- x = 6.5 × 2 / 65 = **0.2 g**

**答**：生成氢气 0.2 g。

### 例题 2（含气体体积）
**题目**：加热分解 15.8 g 高锰酸钾，理论上可制得氧气多少克？标准状况下为多少升？（O₂ 密度 1.429 g/L）

**解答**：
- 设生成 O₂ 质量为 x
- 2KMnO₄ →△→ K₂MnO₄ + MnO₂ + O₂↑
- 相对质量：2×158 = 316 / 32
- 316 / 32 = 15.8 / x
- x = 15.8 × 32 / 316 = **1.6 g**
- 体积 V = m / ρ = 1.6 / 1.429 ≈ **1.12 L**

## 三、题型 2：限量试剂/过量计算（重点）

### 判断方法
**比较法**：将各反应物的实际质量比与方程式中理论质量比对比，**比值小者（相对不足者）为限量试剂**，以此为基准计算。

### 判断口诀
> **"谁不足谁限量，按限量算产物。"**

### 例题 3
**题目**：将 5.6 g 铁片放入 50 g 稀盐酸（溶质质量分数 14.6%）中，充分反应后生成 H₂ 多少克？

**解答**：
- 第一步：判断限量试剂
  - Fe 质量 = 5.6 g
  - HCl 质量 = 50 × 14.6% = 7.3 g
  - 方程式理论比 Fe:HCl = 56:73 = 0.767
  - 实际比 5.6 : 7.3 = 0.767
  - 恰好完全反应（若不等，比值小者为限量）

- 第二步：以 Fe 为基准计算
- Fe + 2HCl → FeCl₂ + H₂↑
- 56 / 2 = 5.6 / x
- x = 5.6 × 2 / 56 = **0.2 g**

**答**：生成 H₂ 0.2 g。

### 例题 4（过量判断）
**题目**：3.2 g 甲醇（CH₃OH）在 9.6 g 氧气中燃烧，生成 CO₂ 多少克？

**解答**：
- 2CH₃OH + 3O₂ →点燃→ 2CO₂ + 4H₂O
- 理论比 CH₃OH : O₂ = 2×32 : 3×32 = 64 : 96 = 2:3
- 实际比 3.2 : 9.6 = 1:3
- CH₃OH 相对不足（限量），以 CH₃OH 为基准
- 64 / 88 = 3.2 / x
- x = 3.2 × 88 / 64 = **4.4 g**

**答**：生成 CO₂ 4.4 g。

## 四、题型 3：纯度/杂质计算

### 核心公式
- **纯度** = 纯物质质量 / 不纯物质量 × 100%
- **纯物质质量** = 不纯物质量 × 纯度
- **不纯物质量** = 纯物质质量 / 纯度

### 解题流程
1. 用纯度把样品质量换算为纯反应物质量
2. 按方程式六步法计算
3. 产物中若需要纯度，反向换算

### 例题 5
**题目**：含 CaCO₃ 80% 的石灰石 12.5 g，与足量稀盐酸反应，生成 CO₂ 多少克？

**解答**：
- m(CaCO₃) = 12.5 × 80% = 10 g（纯物质）
- 设生成 CO₂ 质量为 x
- CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑
- 100 / 44 = 10 / x
- x = 10 × 44 / 100 = **4.4 g**

**答**：生成 CO₂ 4.4 g。

## 五、题型 4：产率计算

### 公式
**产率** = 实际产量 / 理论产量 × 100%

### 例题 6
**题目**：理论上电解 9 g 水可制得 H₂ 1 g，实际实验中只收集到 0.9 g H₂，产率是多少？

**解答**：
- 先验证理论产量：2H₂O →通电→ 2H₂↑ + O₂↑
- 36 / 4 = 9 / x → x = 1 g（与题意一致）
- 产率 = 0.9 / 1 × 100% = **90%**

## 六、题型 5：差量法（重点技巧）

### 原理
利用反应前后**质量差**（或体积差）与方程式中物质的质量成比例关系，简化计算。

### 适用场景
- 金属与酸反应，金属片质量变化
- 气体被吸收后装置质量变化
- 沉淀生成后溶液质量变化

### 例题 7
**题目**：将一根 10.0 g 的铁钉浸入 CuSO₄ 溶液中，充分反应后取出洗净干燥，称得质量为 10.4 g。求有多少克铁参加反应？

**解答**：
- 反应：Fe + CuSO₄ → FeSO₄ + Cu
- 每反应 56 g Fe，析出 64 g Cu，铁钉增重 64 - 56 = 8 g
- 设参加反应的 Fe 质量为 x
- 56 / 8 = x / (10.4 - 10.0)
- 56 / 8 = x / 0.4
- x = 56 × 0.4 / 8 = **2.8 g**

**答**：有 2.8 g 铁参加反应。

## 七、题型 6：元素质量分数计算

### 公式
**某元素质量分数** = (该元素原子数 × 相对原子质量) / 化合物相对分子质量 × 100%

### 例题 8（含氮量对比）
**题目**：计算尿素 CO(NH₂)₂ 和硝酸铵 NH₄NO₃ 的含氮量，比较哪种含氮更高。

**解答**：
- **尿素 CO(NH₂)₂**：
  - 相对分子质量 = 12 + 16 + 2×(14 + 2×1) = 12 + 16 + 32 = **60**
  - 含氮量 = (2×14) / 60 × 100% = 28/60 ≈ **46.7%**
- **硝酸铵 NH₄NO₃**：
  - 相对分子质量 = 14 + 4 + 14 + 3×16 = **80**
  - 含氮量 = (2×14) / 80 × 100% = 28/80 = **35%**

**结论**：尿素含氮量（46.7%）> 硝酸铵含氮量（35%），尿素是含氮量最高的固体氮肥。

### 常见化合物元素质量分数速查
| 化合物 | 相对分子质量 | 关键元素质量分数 |
|--------|:-----------:|-----------------|
| 尿素 CO(NH₂)₂ | 60 | N：46.7% |
| 硝酸铵 NH₄NO₃ | 80 | N：35% |
| 硫酸铵 (NH₄)₂SO₄ | 132 | N：21.2% |
| 碳酸氢铵 NH₄HCO₃ | 79 | N：17.7% |
| 水 H₂O | 18 | H：11.1%，O：88.9% |
| CO₂ | 44 | C：27.3%，O：72.7% |
| Fe₂O₃ | 160 | Fe：70%，O：30% |

## 八、题型 7：混合物计算（压轴）

### 平均值法
利用"平均值必介于两组分之间"的原理，快速判断混合物组成。

### 例题 9
**题目**：某铁粉样品含有少量杂质（可能是 Zn、Cu、C 中的一种或几种），取 5.6 g 样品加入足量稀 H₂SO₄，产生 H₂ 0.2 g。判断杂质可能是什么？

**解答**：
- 假设全为 Fe：56 / 2 = x / 0.2 → x = 5.6 g，恰好等于样品质量
- 若样品含 Zn：Zn 比 Fe 重（同质量产 H₂ 少），样品质量应 > 5.6 g 才能产 0.2 g H₂ → 不符合
- 若样品含 Cu：Cu 不反应，需更多 Fe 才能产 0.2 g H₂ → 若产 0.2 g H₂ 则样品质量 < 5.6 g 不成立（实际等于 5.6 g，说明 Cu 的存在被某种更轻物质抵消）
- 若样品含 C：C 不反应，Fe 量 < 5.6 g，产 H₂ < 0.2 g → 不符合

**结论**：实际产 H₂ 0.2 g = 纯铁理论值，说明样品中杂质对 H₂ 产量无净影响，或样品就是纯铁。**若题目确定有杂质，需重新审查数据**。

## 九、题型 8：图像/表格分析题（压轴）

### 解题要点
1. **看清坐标轴含义**（横坐标：时间/温度/质量；纵坐标：质量/体积/pH）
2. **关键点**：起点、转折点（拐点）、终点、交点
3. **拐点**：表示反应恰好完全，限量试剂耗尽
4. **斜率**：反应速率

### 常见图像类型
| 图像类型 | 横坐标 | 纵坐标 | 拐点意义 |
|----------|--------|--------|----------|
| 金属+酸产 H₂ | 时间 | H₂ 体积 | 金属耗尽 |
| 中和滴定 pH | 滴加酸体积 | pH | 完全中和点 |
| 沉淀质量 | 滴加试剂 | 沉淀质量 | 沉淀完全 |
| 溶解度曲线 | 温度 | 溶解度 | 两物质溶解度相等 |

## 十、解题通用技巧总结

### 10.1 计算前"三查"
1. **查方程式**：化学式正确、配平、条件齐全
2. **查相对分子质量**：乘以计量数后再比
3. **查单位**：题目给的是质量还是体积？是否带纯度？

### 10.2 计算中"三注意"
1. **纯度换算**：题目给样品质量，必须先换算为纯物质质量
2. **限量判断**：两种反应物都给量时，必须先判断限量
3. **比例对应**：物质与质量一一对应，不要串行

### 10.3 计算后"三验证"
1. **结果合理性**：质量是否合理（如 O₂ 产量不可能超过反应物总质量）
2. **单位齐全**：答案带单位
3. **有效数字**：按题目要求保留小数位

## 十一、易错点速记

| 易错点 | 正确做法 |
|--------|----------|
| 直接用样品质量代入方程式 | 必须先换算纯物质质量 |
| 两种反应物都给量，任选一个算 | 必须判断限量试剂 |
| 计算相对分子质量漏乘计量数 | Fe₂O₃ 应算 2×56+3×16=160，不是单分子再乘 |
| 体积/质量混淆 | 用 ρ=m/V 换算，O₂ 密度 1.429 g/L |
| 气体未换算 | 题目给体积时用 ρ 换算为质量再代入 |
| 答案不带单位 | 必须带 g / L / mol 等单位 |

## 十二、综合练习题（附答案）

### 练习 1（限量试剂）
将 10 g 含 CaCO₃ 80% 的石灰石高温煅烧，残留固体 6.8 g，求 CO₂ 体积（标况）。
- **答案**：CaCO₃ 纯质 = 8 g；设生成 CO₂ 质量为 x：100/44 = 8/x → x = 3.52 g；V = 3.52/1.977 ≈ **1.78 L**

### 练习 2（差量法）
11.2 g 铁钉放入 CuSO₄ 溶液，反应后铁钉增重至 12.0 g，求析出铜的质量。
- **答案**：增重 0.8 g；每析出 64 g Cu 增重 8 g；析出 Cu = 0.8×64/8 = **6.4 g**

### 练习 3（纯度+产率）
5.0 g 含杂质（不反应）的锌样品与足量稀硫酸反应，理论上可产 H₂ 0.15 g，实际收集 0.14 g。求锌的纯度与产率。
- **答案**：纯 Zn = 0.15×65/2 = 4.875 g → 纯度 = 4.875/5.0 = **97.5%**；产率 = 0.14/0.15 = **93.3%**

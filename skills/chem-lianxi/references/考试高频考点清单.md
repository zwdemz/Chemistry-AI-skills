# 考试高频考点清单

> 初中化学中考必考/常考/轮考三级分类清单，配合 chem-lianxi skill 出题与诊断。

---

## 一、必考点（每次考试 80%+ 出现）

### 1. 氧气的制取与性质
- [ ] 实验室制 O₂ 三种方法（高锰酸钾/氯酸钾/过氧化氢）
- [ ] 装置选择与气密性检查
- [ ] 收集方法（排水/向上排空气）
- [ ] 验满与检验
- [ ] 催化剂"一变两不变"

### 2. 二氧化碳的制取与性质
- [ ] 实验室制 CO₂（CaCO₃ + 2HCl）
- [ ] 不用稀硫酸原因（CaSO₄ 微溶包裹）
- [ ] 不用 Na₂CO₃ 原因（反应过快）
- [ ] 向上排空气法收集（CO₂ 密度大、溶于水）
- [ ] 验满与检验（澄清石灰水）

### 3. 金属活动性顺序
- [ ] 完整背诵 K Ca Na Mg Al Zn Fe Sn Pb (H) Cu Hg Ag Pt Au
- [ ] 应用规则：前置后、盐溶液、K/Ca/Na 除外
- [ ] Fe + CuSO₄、Cu + 2AgNO₃ 等置换反应
- [ ] 判断金属能否与酸反应（位于 H 前）

### 4. 酸的化学通性（5 条）
- [ ] 酸 + 指示剂（石蕊变红、酚酞不变色）
- [ ] 酸 + 活泼金属 → 盐 + H₂↑
- [ ] 酸 + 金属氧化物 → 盐 + 水
- [ ] 酸 + 碱（中和）→ 盐 + 水
- [ ] 酸 + 盐 → 新酸 + 新盐

### 5. 碱的化学通性（4 条）
- [ ] 碱 + 指示剂（石蕊变蓝、酚酞变红）
- [ ] 碱 + 非金属氧化物 → 盐 + 水
- [ ] 碱 + 酸（中和）
- [ ] 碱 + 盐 → 新碱 + 新盐

### 6. 复分解反应条件
- [ ] 生成物有沉淀/气体/水之一即可
- [ ] Cl⁻ 检验（AgNO₃ + 稀 HNO₃）
- [ ] SO₄²⁻ 检验（BaCl₂ + 稀 HNO₃）
- [ ] CO₃²⁻ 检验（稀 HCl + 澄清石灰水）

### 7. 化学方程式
- [ ] 配平方法（最小公倍数/奇偶法/观察法）
- [ ] 反应条件标注（点燃/加热△/高温/通电/催化剂）
- [ ] ↑↓ 符号规范
- [ ] 60+ 核心方程式（见方程式速查表）

### 8. 溶质质量分数
- [ ] ω = m溶质 / m溶液 × 100%
- [ ] 稀释公式 m₁ω₁ = m₂ω₂
- [ ] 配制步骤（计算/称量/量取/溶解/装瓶）

## 二、常考点（每次考试 60-80% 出现）

### 9. 控制变量法
- [ ] 三要素（控制变量/自变量/因变量）
- [ ] 催化剂种类/用量对比
- [ ] 温度/浓度/表面积对反应速率的影响

### 10. 燃烧与灭火
- [ ] 燃烧三条件（可燃物 + O₂ + 着火点）
- [ ] 灭火原理（破坏任一条件）
- [ ] 爆炸极限（H₂ 4%-74.2%）

### 11. 中和反应
- [ ] H⁺ + OH⁻ → H₂O
- [ ] 指示剂判断终点
- [ ] 应用（改良酸性土壤用熟石灰）

### 12. 铁的锈蚀与防护
- [ ] 条件：Fe + O₂ + H₂O
- [ ] 铁锈特点（Fe₂O₃·xH₂O，疏松多孔）
- [ ] 防护方法（涂油/刷漆/镀金属/合金）

### 13. pH
- [ ] pH < 7 酸、= 7 中、> 7 碱
- [ ] pH 越小酸性越强
- [ ] 酸雨 pH < 5.6

### 14. 电解水
- [ ] 方程式 2H₂O →通电→ 2H₂↑ + O₂↑
- [ ] 正氧负氢、氢二氧一
- [ ] 加稀 H₂SO₄ 目的（增强导电性）

### 15. 实验现象描述
- [ ] 颜色/状态/光/热/气味变化
- [ ] 不能写"说明是__"
- [ ] 反应物消耗 + 生成物出现

### 16. 碳和碳的氧化物
- [ ] 碳单质（金刚石/石墨/C₆₀）
- [ ] CO₂ 性质（不可燃不助燃、与水反应、与碱反应）
- [ ] CO 性质（可燃、还原性、剧毒）

## 三、轮考点（每次考试 30-60% 出现）

### 17. 物理变化 vs 化学变化
- [ ] 判断依据：是否有新物质生成
- [ ] 本质区别：分子是否改变

### 18. 分子与原子
- [ ] 分子性质（小、动、间隔）
- [ ] 化学变化本质（分子分裂为原子，原子重新组合）

### 19. 原子结构
- [ ] 核电荷数 = 质子数 = 核外电子数 = 原子序数
- [ ] 相对原子质量 ≈ 质子数 + 中子数
- [ ] 核外电子排布（前 20 号）

### 20. 元素与化合价
- [ ] 化合价口诀
- [ ] 化学式书写（正前负后，交叉化简）
- [ ] 离子符号书写

### 21. 化学式计算
- [ ] 相对分子质量
- [ ] 元素质量比
- [ ] 元素质量分数

### 22. 质量守恒定律
- [ ] 三不变（原子种类/数目/质量）
- [ ] 微观解释

### 23. 水的净化
- [ ] 顺序：沉淀 → 过滤 → 吸附 → 蒸馏
- [ ] 过滤"一贴二低三靠"
- [ ] 硬水与软水鉴别

### 24. 溶解度
- [ ] 四要素（一定温度/100g 溶剂/饱和状态/质量 g）
- [ ] 溶解度曲线含义
- [ ] 气体溶解度（温度反比、压强正比）

### 25. 化石燃料与清洁能源
- [ ] 煤/石油/天然气（不可再生）
- [ ] 乙醇/氢气（清洁能源）
- [ ] CH₄、C₂H₅OH 燃烧方程式

### 26. 合金
- [ ] 概念（金属 + 金属/非金属，混合物）
- [ ] 生铁与钢（含碳量不同）
- [ ] 特性（熔点低、硬度大）

### 27. 常见盐
- [ ] NaCl/Na₂CO₃/NaHCO₃/CaCO₃ 俗称与用途
- [ ] 纯碱不是碱（Na₂CO₃ 是盐）

### 28. 化学肥料
- [ ] N/P/K 肥作用
- [ ] 铵态氮肥检验（加碱 + 红色石蕊试纸）
- [ ] 不能与碱混合施用

### 29. 营养素
- [ ] 六大营养素（蛋白质/糖类/油脂/维生素/无机盐/水）
- [ ] 有机物 vs 无机物

### 30. 环境保护
- [ ] 酸雨（SO₂、NO₂）
- [ ] 温室效应（CO₂）
- [ ] 白色污染（塑料）

## 四、易错点速记（高频陷阱）

1. **铁的化合价**：与酸/盐反应生成 +2 价亚铁（浅绿色），不是 +3 价（黄色）
2. **稀硫酸代替盐酸**：CaCO₃ + H₂SO₄ 反应被 CaSO₄ 包裹中止
3. **CO₂ 验满 vs 检验**：验满放瓶口、检验伸入瓶中
4. **量筒读数**：仰视读数偏小实际偏多，俯视读数偏大实际偏少
5. **托盘天平**：左物右码（记反了用 m = m砝码 - m游码 调整）
6. **电解水体积比**：H₂:O₂ = 2:1（负极 H₂ 体积是正极 O₂ 的 2 倍）
7. **白磷保存**：保存在水中（着火点 40℃ 低），不能暴露空气中
8. **NaOH 保存**：密封（吸水潮解 + 吸收 CO₂ 变质）
9. **中和反应**：不一定有明显现象，常借助指示剂判断
10. **CO₂ 通入石蕊变红**：是 CO₂ + H₂O → H₂CO₃，不是 CO₂ 本身使石蕊变红

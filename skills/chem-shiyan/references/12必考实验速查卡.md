# 12 必考实验速查卡

> 课标必考 12 个实验的方程式+装置+步骤+采分点速查，一卡一实验。

---

## EXP001 · 高锰酸钾制氧气

- **方程式**：2KMnO₄ →△→ K₂MnO₄ + MnO₂ + O₂↑
- **装置**：固体加热型（试管 + 酒精灯 + 导管 + 水槽 + 集气瓶）
- **关键步骤**：①查气密性 → ②装药品+塞棉花 → ③管口略下倾固定 → ④预热再集中加热 → ⑤气泡均匀时收集 → ⑥先撤导管后熄灯
- **现象**：大量气泡，无色无味气体，带火星木条复燃
- **📌 采分点**：①管口下倾原因 ②塞棉花作用 ③先撤导管后熄灯 ④收集时机 ⑤方程式配平

---

## EXP002 · 石灰石制 CO₂

- **方程式**：CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑
- **装置**：固液不加热型（锥形瓶 + 长颈漏斗 + 导管）
- **关键步骤**：①查气密性 → ②装石灰石 → ③加稀盐酸 → ④向上排空气法收集 → ⑤验满（燃着木条放瓶口熄灭）
- **现象**：石灰石表面大量气泡，通入澄清石灰水变浑浊
- **📌 采分点**：①不用稀硫酸原因（CaSO₄ 包裹）②不用 Na₂CO₃ 原因（过快）③不用排水法（CO₂ 溶于水）④验满位置 ⑤方程式

---

## EXP003 · 金属活动性（Fe + CuSO₄）

- **方程式**：Fe + CuSO₄ → FeSO₄ + Cu
- **装置**：试管 + 砂纸 + 铁丝
- **关键步骤**：①加 CuSO₄ 溶液 → ②砂纸打磨铁丝 → ③浸入溶液 → ④静置观察 → ⑤取出观察
- **现象**：铁丝表面附红色铜，溶液蓝→浅绿
- **📌 采分点**：①打磨原因（除氧化膜）②颜色变化原因（Cu²⁺→Fe²⁺）③金属活动性应用 ④反应类型（置换）

---

## EXP004 · 电解水

- **方程式**：2H₂O →通电→ 2H₂↑ + O₂↑
- **装置**：水电解器 + 直流电源
- **关键步骤**：①加水 + 少量稀 H₂SO₄ → ②接通直流电源 → ③观察电极气泡 → ④正极带火星木条 → ⑤负极燃着木条
- **现象**：两极产生气泡，体积比 H₂:O₂ = 2:1
- **📌 采分点**：①加 H₂SO₄ 目的（增强导电性）②正氧负氢 ③体积比 2:1 ④用直流电 ⑤结论：水由 H、O 组成

---

## EXP005 · 空气中氧气含量测定

- **方程式**：4P + 5O₂ →点燃→ 2P₂O₅
- **装置**：集气瓶 + 燃烧匙 + 导管 + 烧杯
- **关键步骤**：①止水夹夹紧导管 → ②点燃红磷迅速伸入集气瓶 → ③冷却至室温 → ④打开止水夹 → ⑤观察进水量
- **现象**：大量白烟，冷却后水进入约 1/5 体积
- **📌 采分点**：①红磷过量原因 ②冷却至室温原因 ③不能用 C/S 代替（生成气体）④结论：O₂ 占 1/5 ⑤气密性重要性

---

## EXP006 · 粗盐提纯

- **核心操作**：溶解 → 过滤 → 蒸发 → 称量
- **装置**：烧杯 + 玻璃棒 + 漏斗 + 蒸发皿 + 酒精灯 + 托盘天平
- **玻璃棒三次作用**：①溶解时搅拌 ②过滤时引流 ③蒸发时搅拌防飞溅
- **📌 采分点**：①四次玻璃棒作用 ②过滤一贴二低三靠 ③蒸发停止时机（出现较多固体）④产率计算 ⑤损失原因分析

---

## EXP007 · 一定质量分数溶液配制

- **核心步骤**：计算 → 称量 → 量取 → 溶解 → 装瓶贴签
- **装置**：托盘天平 + 量筒 + 烧杯 + 玻璃棒 + 试剂瓶
- **示例**：配制 50 g 6% NaCl 溶液 → NaCl 3.0 g + 水 47 mL
- **📌 采分点**：①计算准确 ②称量规范 ③量取读数（平视）④溶解搅拌 ⑤装瓶贴签 ⑥误差分析（仰视/俯视）

---

## EXP008 · 酸碱中和反应

- **方程式**：HCl + NaOH → NaCl + H₂O
- **装置**：烧杯 + 胶头滴管 + 玻璃棒 + 指示剂（酚酞）
- **关键步骤**：①NaOH 溶液 + 酚酞（变红）→ ②逐滴加 HCl 搅拌 → ③颜色恰好消失时停止
- **现象**：红色逐渐变浅，最后无色
- **📌 采分点**：①指示剂选择 ②逐滴加入原因 ③搅拌作用 ④恰好褪色（不多不少）⑤放热现象

---

## EXP009 · 燃烧条件探究

- **装置**：烧杯 + 白磷 + 红磷 + 铜片 + 热水
- **关键步骤**：①铜片放白磷、红磷 → ②热水放白磷 → ③观察三者现象
- **现象**：①铜片白磷燃烧 ②铜片红磷不燃烧（着火点高）③水中白磷不燃烧（无 O₂）④通入 O₂ 后水中白磷燃烧
- **📌 采分点**：①控制变量法 ②三条件缺一不可 ③对比白磷/红磷着火点 ④热水白磷通 O₂ 后燃烧结论

---

## EXP010 · 铁钉锈蚀条件

- **装置**：3 支试管 + 铁钉 + 水 + 油 + 干燥剂
- **设置**：①试管 1：水+敞口（部分浸入）②试管 2：水煮沸+油封（无 O₂）③试管 3：干燥剂密封（无水）
- **现象**：3-5 天后，试管 1 生锈，试管 2/3 不生锈
- **📌 采分点**：①变量控制 ②水煮沸目的（除 O₂）③油封目的（隔绝 O₂）④结论：Fe+O₂+H₂O 同时存在

## EXP011 · 一定溶质质量分数溶液配制（精细版）

- 与 EXP007 类似，重点是仪器读数与误差分析
- **📌 采分点**：①计算 ②称量（左物右码）③量取（平视）④溶解（玻璃棒）⑤装瓶 ⑥误差判断

## EXP012 · 常见酸和碱的性质

- **酸性质实验**：HCl/H₂SO₄ + 指示剂/金属/金属氧化物/碱/盐
- **碱性质实验**：NaOH/Ca(OH)₂ + 指示剂/非金属氧化物/酸/盐
- **📌 采分点**：①酸的 5 条通性 ②碱的 4 条通性 ③指示剂颜色变化 ④中和反应 ⑤浓硫酸特性

## 实验装置选择速查

| 类型 | 适用反应 | 示例 |
|------|----------|------|
| **固体加热型** | 固 + 固，加热 | 高锰酸钾制 O₂ |
| **固液不加热型** | 固 + 液，不加热 | H₂O₂ + MnO₂ 制 O₂；石灰石 + HCl 制 CO₂ |
| **固液加热型** | 固 + 液，加热 | 初中较少 |
| **液液加热型** | 液 + 液，加热 | 中和反应（需控温）|

---

## 收集方法选择

| 方法 | 适用条件 | 示例 |
|------|----------|------|
| **排水法** | 不溶水不反应 | O₂、H₂、CO |
| **向上排空气法** | 密度大于空气 | O₂、CO₂ |
| **向下排空气法** | 密度小于空气 | H₂、NH₃ |

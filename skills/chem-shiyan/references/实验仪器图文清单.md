# 实验仪器图文清单

> 初中化学常用仪器 7 大类 30+ 种，含用途、规格与使用注意事项。

---

## 一、加热类仪器

| 仪器 | 主要用途 | 使用注意事项 |
|------|----------|-------------|
| **酒精灯** | 加热（外焰温度 500℃）| ①酒精量 1/4~2/3 ②禁止对火/吹灭 ③用外焰加热 ④先熄灭再添加酒精 |
| **酒精喷灯** | 高温加热（1000℃+）| 初中较少使用 |
| **试管** | 少量试剂反应、加热 | ①液体不超 1/3 ②加热管口不对人 ③预热防炸裂 ④固体加热管口略下倾 |
| **蒸发皿** | 蒸发、浓缩溶液 | ①可直接加热 ②加热后用坩埚钳取 ③冷却时放在石棉网上 |
| **燃烧匙** | 盛放固体在气体中燃烧 | ①铜质或铁质 ②伸入集气瓶中点燃 |
| **石棉网** | 垫在烧杯下加热 | 均匀受热，防止烧杯炸裂 |
| **坩埚** | 高温灼烧固体 | ①泥三角支撑 ②坩埚钳夹取 ③冷却时防止骤冷炸裂 |
| **泥三角** | 支撑坩埚加热 | 配合坩埚使用 |

## 二、量取类仪器

| 仪器 | 规格/精度 | 用途 | 注意事项 |
|------|-----------|------|----------|
| **量筒** | 10mL/0.1mL、50mL/1mL、100mL/1mL | 量取液体体积 | ①不能加热 ②不能配溶液 ③不能在其中反应 ④读数平视凹液面 |
| **托盘天平** | 0.1 g | 称量固体质量 | ①左物右码 ②称量纸（易潮解用玻璃器皿）③称前调零 ④不能称热物 |
| **温度计** | 1℃ | 测温度 | ①玻璃泡不触底 ②不能当搅拌棒 ③读数时液泡仍在液中 |
| **胶头滴管** | ≈ 0.05 mL/滴 | 滴加少量液体 | ①竖直悬空 ②不伸入容器 ③不倒置 ④用完即洗 ⑤不一管多用 |
| **量气装置** | 测气体体积 | 排水法读水量 | 初中较少使用 |

## 三、容器类仪器

| 仪器 | 用途 | 注意事项 |
|------|------|----------|
| **烧杯** | 配溶液、大量反应 | ①加热垫石棉网 ②溶解时用玻璃棒搅拌 |
| **锥形瓶** | 滴定、反应容器 | 加热垫石棉网 |
| **集气瓶** | 收集气体 | ①磨砂面朝下盖玻璃片 ②铁丝燃烧瓶底放水/沙 |
| **广口瓶** | 盛放固体试剂 | 配毛玻璃盖 |
| **细口瓶** | 盛放液体试剂 | 标签向外 |
| **滴瓶** | 盛放少量液体 | 专用滴管不清洗 |
| **水槽** | 排水法收集气体 | 排水法专用 |

## 四、分离类仪器

| 仪器 | 用途 | 注意事项 |
|------|------|----------|
| **漏斗** | 过滤 | 一贴二低三靠 |
| **长颈漏斗** | 固液反应装置加液 | 下端伸入液面下（防气体逸出）|
| **分液漏斗** | 控制加液速度 | 初中较少使用 |
| **玻璃棒** | 引流、搅拌、蘸取 | ①引流靠三层滤纸 ②搅拌不碰壁 ③蘸取滴在试纸上 |

---

## 五、夹持/支撑类仪器

| 仪器 | 用途 | 注意事项 |
|------|------|----------|
| **铁架台** | 固定装置 | 铁夹夹试管中上部 |
| **铁圈** | 支撑漏斗/蒸发皿 | 配合铁架台使用 |
| **铁夹** | 夹持试管 | 夹中上部（距管口 1/3）|
| **试管夹** | 加热时夹试管 | 从底往上套 |
| **坩埚钳** | 夹坩埚/蒸发皿 | 预热后使用 |
| **镊子** | 取块状固体 | 一横二放三慢竖 |
| **药匙** | 取粉末固体 | 一斜二送三直立 |
| **纸槽** | 送粉末入试管底 | 自制即可 |
| **三脚架** | 支撑加热 | 配合石棉网/蒸发皿 |

## 八、洗涤标准

### 玻璃仪器洗净标准
内壁附着的水**既不聚成水滴也不成股流下**。

### 特殊污垢洗涤
| 污垢 | 洗涤剂 | 原理 |
|------|--------|------|
| 油污 | 热纯碱溶液/洗涤剂 | 乳化/皂化 |
| 铁锈 | 稀盐酸 | Fe₂O₃ + 6HCl → 2FeCl₃ + 3H₂O |
| 碳酸盐（水垢）| 稀盐酸 | CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑ |
| 二氧化锰 | 浓盐酸加热 | MnO₂ + 4HCl（浓）→△→ MnCl₂ + Cl₂↑ + 2H₂O |
| 银镜 | 稀硝酸 | Ag + 稀 HNO₃（初中较少用）|
| 碘 | 酒精 | 相似相溶 |
